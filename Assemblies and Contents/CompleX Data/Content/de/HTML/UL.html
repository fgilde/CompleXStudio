<UL>...</UL>

HTML-End-Tag: Erforderlich  

Das Element ul ist ein Container für eine ungeordnete Elementliste. Ein "ungeordnete Liste" bedeutet, dass den Einträgen ein Symbol vorangestellt wird (abhängig von der Einstellung des Attributs type oder des Stylesheet-Attributs list-style-type), das außer durch die Position innerhalb der Liste keine bestimmte Reihenfolge der Einträge impliziert. Der Inhalt für die einzelnen Listeneinträge wird durch ein verschachteltes li-Element definiert. Wenn Sie eine Stylesheet-Regel auf ein ul-Element anwenden, wird der Stil von den verschachtelten li-Element geerbt. (In Navigator 4 ist gelegentlich ein etwas merkwürdiges Verhalten zu beobachten.)

 
Beispiel
  
<UL>
    <LI>Africa</LI>
    <LI>Antarctica</LI>
    <LI>Asia</LI>
    <LI>Australia</LI>
    <LI>Europe</LI>
    <LI>North America</LI>
    <LI>South America</LI>
</UL>

compactNN 6 IE 4 HTML 3.2  

compact

Optional  

Ein Boolesches Attribut, das ursprünglich den Zweck hatte, eine kompaktere Darstellung von Listen in Browsern zu ermöglichen (mit einem geringeren Zeilenabstand zwischen den Elementen). Das Attribut compact wird aus Gründen der HTML-Kompatibilität zwar als unterstütztes Attribut aufgeführt, hat aber in Mainstream-Browsern keinerlei Wirkung. Zur Steuerung der Elementgröße und des Zeilenstands sollten Sie Stylesheets verwenden.

 
Beispiel
  
<UL compact>...</UL>

Ist dieses Attribut vorhanden, hat es den Wert true.

false
langNN 3 IE 4 HTML 4  

lang="languageCode"

Optional  

Die für die Attributwerte und Inhalte des Elements verwendete Sprache. Mithilfe dieser Informationen ist der Browser besser in der Lage, bestimmte Details wie Ligaturen (sofern von der betreffenden Schrift unterstützt oder in der geschriebenen Sprache erforderlich), Anführungszeichen und Worttrennungen korrekt darzustellen. Andere Anwendungen und Suchmaschinen können anhand dieser Informationen Wörterbücher für die Rechtschreibprüfung auswählen und Indizes erstellen.

 
Beispiel
  
<SPAN lang="de">Deutsche Bundesbahn</SPAN>
 
Wert

Sprachcode, bei dem nicht zwischen Groß- und Kleinschreibung unterschieden wird.
 
Standardwert

Vorgabe des Browsers.

typeNN alle IE alle HTML 3.2  

type="labelType"

Optional  

Das Attribut type ermöglicht eine flexiblere Darstellung des vorangestellten Symbols oder der Folgenummer im Browser. Sie können angeben, ob das vorangestellte Symbol ein gefüllter Kreis, ein leerer Kreis oder ein Quadrat sein soll. Ein gefüllter Kreis ist auch als Listenpunkt, Aufzählungszeichen oder Bullet bekannt. Das Quadrat wird in frühen Macintosh-Browsern unausgefüllt dargestellt, in Windows und modernen Browsern für alle Betriebssysteme ausgefüllt. In HTML 4 wird dem Stylesheet-Attribut list-style-type Vorrang vor dem Attribut type eingeräumt.

 
Beispiel
  
<UL type="disc">...</UL>
 
Wert

Mögliche Werte: circle | disc | square.

 
Standardwert

disc

[window.]document.getElementById(elementID).type
