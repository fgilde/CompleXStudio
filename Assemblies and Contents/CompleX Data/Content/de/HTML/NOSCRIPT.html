HTML-End-Tag: Erforderlich  

Das Element noscript dient zur Anzeige von Inhalt, wenn ein Browser nicht für die Ausführung der in das aktuelle Dokument eingebetteten Skripts konfiguriert ist. In der Praxis wird dieses Element erst ab Navigator 3 und Internet Explorer 4 korrekt eingesetzt. Wurde die Skriptausführung im Browser deaktiviert, wird der Inhalt des Elements noscript an der Stelle dargestellt, an der es sich im Quellcode befindet. Bei älteren und nichtskriptfähigen Browsern wird das Element noscript ignoriert; sein Inhalt wird im Kontext des nächstäußeren Containers dargestellt. Die HTML 4.0-Spezifikation empfiehlt, dass der Inhalt des Elements noscript auch dann in Browsern dargestellt wird, wenn die Skripts weiter vorne im Dokument einen Sprachtyp aufweisen, der vom Browser nicht unterstützt wird oder der nicht aktiviert ist. Sollte ein mit HTML 4 kompatibler Browser entwickelt werden, der keinerlei Skriptfunktionen bietet, sollte der Inhalt des Elements noscript in diesem Browser ebenfalls dargestellt werden.

Alle Attribute des Elements noscript wurden hinzugefügt, um Cascading Style Sheets (CSS), Internationalisierung und Ereignisse für HTML 4 zu unterstützen.

 
Beispiel
  
<NOSCRIPT>
This document contains programming that requires a scriptable browser, such
as Microsoft Internet Explorer or Netscape Navigator. You may not have full
access to this page's powers at this time.
</NOSCRIPT>

lang="languageCode"

Optional  

Die für die Attributwerte und Inhalte des Elements verwendete Sprache. Mithilfe dieser Informationen ist der Browser besser in der Lage, bestimmte Details wie Ligaturen (sofern von der betreffenden Schrift unterstützt oder in der geschriebenen Sprache erforderlich), Anführungszeichen und Worttrennungen korrekt darzustellen. Andere Anwendungen und Suchmaschinen können anhand dieser Informationen Wörterbücher für die Rechtschreibprüfung auswählen und Indizes erstellen.

 
Beispiel
  
<SPAN lang="de">Deutsche Bundesbahn</SPAN>
 
Wert

Sprachcode, bei dem nicht zwischen Groß- und Kleinschreibung unterschieden wird.
