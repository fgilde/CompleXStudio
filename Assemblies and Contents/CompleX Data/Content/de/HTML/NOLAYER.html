HTML-End-Tag: Erforderlich  

Navigator 4 bietet ein Tag zum Isolieren von Informationsinhalt, der in Browsern angezeigt wird, die das Element layer nicht unterstützen. Der gesamte Inhalt zwischen dem Start- und End-Tag des Elements nolayer wird in Navigator 4 nicht dargestellt. In anderen Browsern wird der Inhalt dagegen dargestellt (hier wird das Tag, aber nicht der Inhalt ignoriert). Sie können das Element nolayer an einer beliebigen Stelle platzieren. Beachten Sie jedoch, dass es nicht wie das Element layer positioniert wird. Netscape 6 (ein Browser, der Ebenen nicht unterstützt) zeigt den Inhalt des nolayer-Elements an.

Für dieses Element gibt es keine Attribute. Wenn Sie Stylesheet-Regeln für das Element nolayer definieren, werden sie von Browsern wie Internet Explorer ignoriert. Sie können das Element nolayer jedoch innerhalb eines div- oder span-Elements verschachteln, um Stylesheet-Regeln mit dem Informationstext zu verknüpfen.

 
Beispiel
  
<LAYER bgcolor="yellow" src="instrux.html" width=200 height=300></LAYER>
