HTML-End-Tag: Erforderlich  

Das Element noframes enthält HTML-Code, der von Browsern dargestellt wird, die keine Frames unterstützen. Browser, die Frames unterstützen, ignorieren sowohl das Element noframes als auch seinen gesamten Inhalt. Der Inhalt dieses Elements sollte die Benutzer über die Verwendung von Frames informieren oder einen Hyperlink zu einer Version der gleichen Seite ohne Frames bereitstellen. Das Element noframes befindet sich in den meisten Fällen innerhalb eines frameset-Elements. Gemäß der HTML 4-Spezifikation ist es jedoch auch ohne weiteres möglich, das Element in einem dargestellten Dokument einzubetten, sollte dies für Ihre konkrete Anwendung sinnvoll sein. noframes kann sinnvoll sein, wenn auf Ihrer Seite ein iframe-Element verwendet wird und Browser, die dieses Element nicht verarbeiten können, Benutzer über den nicht anzeigbaren Inhalt informieren sollen.

Alle Standardattribute des Elements noframes wurden hinzugefügt, um Cascading Style Sheets (CSS) zu unterstützen. Dies scheint widersprüchlich zu sein, da es sehr unwahrscheinlich ist, dass ein Browser zwar CSS unterstützt, aber keine Frames (mit Ausnahme der fehlenden iframe-Unterstützung in Navigator 4).

 
Beispiel
  
<FRAMESET cols="150,*">
    <FRAME name="navbar" src="nav.html">
    <FRAME name="main" src="page1.html">
    <NOFRAMES>Your browser does not support frames. 
    Click <A href="noFramesIndex.html">here</A> for a frameless version.
    </NOFRAMES>
</FRAMESET>

Optional  

Die für die Attributwerte und Inhalte des Elements verwendete Sprache. Mithilfe dieser Informationen ist der Browser besser in der Lage, bestimmte Details wie Ligaturen (sofern von der betreffenden Schrift unterstützt oder in der geschriebenen Sprache erforderlich), Anführungszeichen und Worttrennungen korrekt darzustellen. Andere Anwendungen und Suchmaschinen können anhand dieser Informationen Wörterbücher für die Rechtschreibprüfung auswählen und Indizes erstellen.

 
Beispiel
  
<SPAN lang="de">Deutsche Bundesbahn</SPAN>
 
Wert

Sprachcode, bei dem nicht zwischen Groß- und Kleinschreibung unterschieden wird.
