Mit dem Element i, einem von mehreren Schriftstilelementen in HTML 4, wird der betreffende Inhalt in der Kursivversion der Schrift des nächstäußeren HTML-Containers dargestellt. Sie können verschiedene Stile miteinander kombinieren, indem Sie mehrere Schriftelemente verschachteln, wie z. B. fett kursiv (<B><I>fett und kursiv formatierter Text</I></B>).

Der Browser kann entweder das Kursivformat einer Systemschrift darstellen oder eine Kursivversion der aktuell angegebenen Schrift laden. Um eine optimale Schriftdarstellung zu erreichen, sollten Sie nach Möglichkeit Stylesheets (und eventuell herunterladbare Schriften) verwenden, um eine echte Kursivschriftfamilie anzugeben, anstatt zu riskieren, dass der Browser die Kursivformatierung aus einer Systemschrift extrapoliert.

Sie können die Containerfähigkeit dieses Elements nutzen, indem Sie einigen oder allen i-Elementen auf einer Seite Stylesheet-Regeln zuweisen. Beispielsweise können alle i-Elemente rot dargestellt werden. Durch Zuweisen der Stylesheet-Regel i{color:red} besteht die Möglichkeit, sämtliche Elemente mit nur wenig Code zu formatieren.

Auch wenn dieses Element in HTML 4 weiterhin unterstützt wird, wäre es nicht überraschend, wenn es in Zukunft vom Stylesheet-Attribut font-style verdrängt wird.

 
Beispiel
  
<P>This product is <I>new</I> and <I>improved</I>!</P>

lang="languageCode"

Optional  

Die für die Attributwerte und Inhalte des Elements verwendete Sprache. Mithilfe dieser Informationen ist der Browser besser in der Lage, bestimmte Details wie Ligaturen (sofern von der betreffenden Schrift unterstützt oder in der geschriebenen Sprache erforderlich), Anführungszeichen und Worttrennungen korrekt darzustellen. Andere Anwendungen und Suchmaschinen können anhand dieser Informationen Wörterbücher für die Rechtschreibprüfung auswählen und Indizes erstellen.

 
Beispiel
  
<SPAN lang="de">Deutsche Bundesbahn</SPAN>
