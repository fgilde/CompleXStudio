Das Element code gehört zu einer großen Gruppe von Elementen, die in der HTML 4-Empfehlung als Phrasenelemente bezeichnet werden. Diese Elemente weisen einem bestimmten Teil des Dokuments eine strukturelle Bedeutung zu. Das code-Element wird vor allem dazu verwendet, um ein oder mehrere Inline-Zeichen als Computercode darzustellen (Programmanweisungen, Variablennamen, Schlüsselwörter usw.).

Es liegt am jeweiligen Browser, ob bzw. wie der Inhalt des Elements code vom Rest des body-Elements abgehoben wird. Sowohl Navigator als auch Internet Explorer geben code in einer nichtproportionalen Schrift wieder, normalerweise etwas kleiner als die Standardschrift (bei IE 4 für Macintosh keine kleinere Schrift). Sie können diese Standardeinstellung jedoch mithilfe eines Stylesheets außer Kraft setzen.

Leerraum (einschließlich Wagenrücklaufzeichen) im Inhalt von code-Elementen wird genauso behandelt wie im Inhalt von body-Elementen. Zeilenumbrüche müssen manuell mit br-Elementen eingefügt werden. In der Beschreibung des Elements pre wird erläutert, wie vorformatierter Text unter Beibehaltung des gesamten Leerraums im Quellcode dargestellt werden kann.

 
Beispiel
  
<P>Initialize a variable in JavaScript with the <CODE>var</CODE> keyword.</P>

Optional  

Die für die Attributwerte und Inhalte des Elements verwendete Sprache. Mithilfe dieser Informationen ist der Browser besser in der Lage, bestimmte Details wie Ligaturen (sofern von der betreffenden Schrift unterstützt oder in der geschriebenen Sprache erforderlich), Anführungszeichen und Worttrennungen korrekt darzustellen. Andere Anwendungen und Suchmaschinen können anhand dieser Informationen Wörterbücher für die Rechtschreibprüfung auswählen und Indizes erstellen.

 
Beispiel
  
<SPAN lang="de">Deutsche Bundesbahn</SPAN>
