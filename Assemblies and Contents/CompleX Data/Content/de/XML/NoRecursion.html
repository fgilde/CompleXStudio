Keine Rekursion  
 
Text aus der Spezifikation

Eine analysierte Entity darf weder direkt noch indirekt eine rekursive Referenz auf sich selbst enthalten.
 
Erklärung

Achten Sie darauf, wie Sie Ihre Entities strukturieren. Stellen Sie sicher, dass sie nicht versehentlich eine Kreisreferenz bilden:

<!ENTITY a "&b;">
<!ENTITY b "&c;">
<!ENTITY c "&a;"> <!--wrong!-->
