ID  
 
Text aus der Spezifikation

Werte des Typs ID müssen zur Produktion Name passen. Ein Name darf nicht mehr als einmal als Wert dieses Typs in einem XML-Dokument erscheinen. Das heißt, ID-Werte müssen die Elemente, die sie tragen, eindeutig identifizieren.
 
Erklärung

Die Werte eines Attributs, das als ID deklariert worden ist, müssen sich unterscheiden, d. h keine zwei Werte dürfen denselben Wert besitzen. Diese Einschränkung ist nicht nur auf den Elementtyp beschränkt, sondern gilt für das gesamte Dokument.
