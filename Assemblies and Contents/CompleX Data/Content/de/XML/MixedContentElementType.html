Elementtyp "Gemischter Inhalt"   
<!ELEMENT name (#PCDATA [ | name]+)*>
<!ELEMENT name (#PCDATA)>

Elemente, die mit dem Token #PCDATA deklariert werden, können gemischten Inhalt enthalten, d. h., Text sowie weitere verschachtelte Elemente, die im optionalen Teil der Elementdeklaration deklariert werden. Wird das Token #PCDATA verwendet, so können Sie nicht bestimmen, wie oft andere verschachtelte Elemente mit den analysierbaren Zeichendaten gemischt werden. Soll das Element nur Text aufnehmen, ist das Sternchen optional.
