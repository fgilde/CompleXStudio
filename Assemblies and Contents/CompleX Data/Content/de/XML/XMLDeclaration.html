XML-Deklaration   
<?xml version="version_number" [encoding="encoding-name"][standalone="yes|no"]?>

Die XML-Deklaration dient verschiedenen Zwecken. Sie teilt dem Parser mit, welche Version der XML-Spezifikation verwendet worden ist, in welchem Zeichensatz das Dokument kodiert worden ist und ob das Dokument eigenständig ist oder nicht.

Wenn Sie ein XML-Dokument erstellen müssen, das XML-1.1-Features nutzt, müssen Sie das Pseudoattribut version auf 1.1 setzen:

<?xml version="1.1"?>

Wenn vorhanden, muss die XML-Deklaration das erste sein, was in einem XML-Dokument erscheint. Nichts, außer vielleicht einer Marke für die Unicode-Byte-Ordnung, darf vor dem öffnenden < dieser Struktur stehen. Ist keine XML-Deklaration vorhanden, wird vorausgesetzt, dass das Dokument die XML-1.0-Empfehlung verwendet.
