NOTATION-Attributtyp   
... NOTATION (notation [| notation]*) ...

Der Attributmechanismus NOTATION gestattet XML-Dokument-Autoren festzulegen, dass der Zeicheninhalt bestimmter Elemente den Regeln einer anderen formalen Sprache unterliegt (nicht XML). Das nachfolgende kurze Beispieldokument zeigt, wie Notationen dafür verwendet werden können, den Typ der im Element code_fragment gespeicherten Programmiersprache festzulegen:

<?xml version="1.0"?>
<!DOCTYPE code_fragment
[
<!NOTATION java_code PUBLIC "Java source code">
<!NOTATION c_code PUBLIC "C source code">
<!NOTATION perl_code PUBLIC "Perl source code">
<!ELEMENT code_fragment (#PCDATA)>
<!ATTLIST code_fragment
          code_lang NOTATION (java_code | c_code | perl_code) #REQUIRED>
     
]>
<code_fragment code_lang="c_code">
    main( ) { printf("Hello, world."); }
</code_fragment>
