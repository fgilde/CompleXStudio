Platzhalter-Grafik mit gültigem ALT-Text  
Section 508 1194.22(a); WAI / WCAG 1.0 checkpoint 1.1
 
 
Problembeschreibung

Grafiken in diesem Dokument, die anscheinend nur zur Dekoration dienen (Platzhalter), haben keine entsprechenden gültigen Textbeschreibungen. Eine gültige Textbeschreibung wäre ein Leerzeichen oder eine leere Zeichenfolge für den ALT-Attributwert.

 
 
Problemlösung

Fügen Sie das ALT-Attribut in das IMG-Tag ein.

Für Platzhalter sollte der ALT-Attributwert ein Leerzeichen (" ") oder eine leere Zeichenfolge ("") sein, er sollte jedoch auf jeden Fall vorhanden sein. Das Gleiche gilt für Grafiken, die für Aufzählungen verwendet werden.

 
 
Problemerklärung

Das ALT-Attribut beschreibt die entsprechende Grafik, damit auch Benutzer, die Grafiken im Browser deaktiviert haben, alle Navigationsoptionen der Seite verwenden können. Grafiken, die nur zur Dekoration dienen, benötigen keine Textbeschreibung, da Benutzer, die die Grafik nicht sehen können, es einfach ignorieren würden.

Darüber hinaus ignorieren Browser mit Sprachausgabe oder Bildschirmlesegeräte ALT-Text, der Leerzeichen oder eine leere Zeichenfolge enthält. Eine Zeichenfolge mit Leerzeichen ist eine Zeichenfolge, die ein oder mehrere Leerzeichen enthält (z. B. " "). Eine leere Zeichenfolge ist eine Zeichenfolge ohne jegliches Leerzeichen (z. B. "").

Das ALT-Attribut sollte in jedem Fall definiert werden, selbst wenn als Wert ein Leerzeichen oder eine leere Zeichenfolge eingegeben wird. Ansonsten könnten Bildschirmlesegeräte oder Browser mit Sprachausgabe den Benutzer ablenken, indem sie beispielsweise den Namen der Grafikdatei nennen.
