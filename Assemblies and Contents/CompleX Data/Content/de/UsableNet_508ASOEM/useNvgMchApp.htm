Navigationsmechanismen in konsistenter Weise verwenden  
WAI / WCAG 1.0 Priorität 2 Checkpunkt 13.4
 
 
Problembeschreibung

Die Seite enthält anscheinend eine Navigationsleiste. Stellen Sie sicher, dass Navigationsleisten auf einer Site konsistent gestaltet sind.

 
 
Überprüfung

Sehen Sie sich alle Navigationsleisten in einer Site an und stellen Sie sicher, dass sie so konsistent wie möglich angeordnet, positioniert, formatiert und beschriftet sind.

 
 
Problemerklärung

Konsistenz ist der Schlüssel für eine zugängliche und verwendbare Site. Wenn eine bestimmte Funktion (wie z. B. eine Navigationsleiste) auf allen Seiten einer Site konsistent gestaltet sind, erkennt der Benutzer die Navigationsleiste und deren Schaltflächen/Hyperlinks sofort und kann sie ohne Verzögerung verwenden.

ändert sich eine Hyperlinkgruppe dagegen von Seite zu Seite, muss der Benutzer deren Verwendung jedes Mal neu durchschauen. Dies ist zeitraubend und anstrengend für den Benutzer, außerdem steigt die Fehleranfälligkeit (der Benutzer klickt unter Umständen auf einen falschen Hyperlink).
