INPUT mit gültigem ALT-Attribut  
Section 508 1194.22(a); WAI / WCAG 1.0 checkpoint 1.1
 
 
Problembeschreibung

Eine als Schaltfläche im Formular verwendete Grafik hat keine gültige Textbeschreibung.

Die Grafik hat kein LONGDESC-Attribut, in dem auf eine HTML-Datei mit einer detaillierten, für das ALT-Attribut zu langen Beschreibung verwiesen wird. Eine gültige Beschreibung ist eine Zeichenfolge im ALT-Attribut, bei der es sich nicht um folgende Zeichenfolgen handeln darf:

  • eine leere Zeichenfolge ("")
  • Eine Zeichenfolge mit einem oder mehreren Leerzeichen (" ")
  • Den Namen der Grafikdatei
  • Die Größe der Grafikdatei
  • Eine Zeichenfolge mit mehr als 150 Zeichen (dies ist eine Empfehlung und keine Anforderung gemäß WCAG 1.0 oder 508)
 
 
Problemlösung

Fügen Sie dem INPUT-Tag das ALT-Attribut unter Berücksichtigung folgender Gesichtspunkte hinzu:

  • Die Beschreibung sollte die Auswirkungen erklären, die ein Klick auf die Schaltfläche hat.
  • ALT-Beschreibungen werden von Browsern nicht interpretiert und dürfen keine HTML-Tags enthalten. Eingebettete Tags verwirren sowohl Benutzer als auch manche Suchmaschinen.
  • Zu lange ALT-Beschreibungen werden vom Browser möglicherweise abgeschnitten und erhöhen die Downloadzeit der Seite. Als Faustregel haben sich maximal 10 Wörter bzw. 64 Zeichen bewährt.

Beachten Sie, dass beim Verwenden einer Maus zum Klicken auf die Grafik das Formular und die Koordinaten des Klicks an den Server übertragen werden. Wenn der Server je nach Position des Klicks unterschiedliche Aktionen durchführt, sind Benutzer von Textbrowsern im Nachteil. Daher sollten Sie folgende alternative Lösungen in Betracht ziehen:

  • Verwenden Sie einfache Schaltflächen zum Absenden (jeweils mit eigener Grafik) an Stelle einer einzigen grafischen Schaltfläche. Diese Schaltflächen können Sie über Stylesheets positionieren.
  • Verwenden Sie eine Client-Imagemap in Verbindung mit einem Skript.
 
 
Problemerklärung

Das ALT-Attribut beschreibt die entsprechende Grafik, damit auch Benutzer, die Grafiken im Browser deaktiviert haben, alle Navigationsoptionen der Seite verwenden können. Seit der Einführung von Browsern für PDAs und andere Handgeräte, die nur Text unterstützen, sind die ALT-Beschreibungen wichtiger denn je geworden.

Wenn für die Schaltflächengrafik keine Textbeschreibung angegeben ist, haben Benutzer von Textbrowsern keine Ahnung, was beim Klicken auf die Schaltfläche geschieht.
