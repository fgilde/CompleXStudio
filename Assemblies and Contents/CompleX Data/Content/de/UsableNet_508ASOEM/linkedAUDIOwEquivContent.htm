Verknüpftes AUDIO-Objekt mit äquivalentem CONTENT  
Section 508 1194.22(a); WAI / WCAG 1.0 checkpoint 1.1
 
 
Problembeschreibung

Die Seite enthält A- oder AREA-Elemente, die auf Audiodateien verweisen.

Stellen Sie sicher, dass die Audiodatei im Dokument beschrieben wird oder das Dokument einen Hyperlink zu einer Transkriptionsdatei enthält, in der die Inhalte der Audiodatei beschrieben werden.

 
 
Überprüfung

Fügen Sie eine aussagekräftige Beschreibung der Audiodatei ein.
Stellen Sie insbesondere sicher, dass die Seite eines der folgenden Elemente enthält:

  • Eine Transkription der Audio-/Videodatei
  • Einen Hyperlink, der auf eine Transkription verweist
 
 
Problemerklärung

Textäquivalente für Audiodateien können als Textlinks auf eine Transkription bzw. eine Beschreibung der Audiodatei bereitgestellt werden. Der Hyperlink zur Transkription sollte sich an einer gut sichtbaren Stelle, z. B. oben auf der Seite, befinden.

Wenn keine Transkriptionen vorhanden sind, können folgende Benutzer die Audioausgabe nicht hören und deren Inhalte nicht nutzen:

  • Taube oder schwerhörige Benutzer
  • Benutzer mit Audiogeräten in lauten Umgebungen
  • Benutzer, die nicht über das entsprechende Plugin zum Abspielen von Audiodateien verfügen
  • Benutzer, die die Seite über einen Computer ohne Audiogeräte aufrufen
  • Benutzer, die die Seite an Orten aufrufen, an denen keine Audiodaten abgespielt werden dürfen, z. B. in öffentlichen Bibliotheken
