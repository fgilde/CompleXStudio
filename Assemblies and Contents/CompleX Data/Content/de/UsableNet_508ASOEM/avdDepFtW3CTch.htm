überholte Features von W3C-Technologien vermeiden  
WAI / WCAG 1.0 Priorität 2 Checkpunkt 11.2
 
 
Problembeschreibung

Die Seite enthält Tags, die in HTML 4.01 überholt sind. Diese sollten nicht verwendet werden.

 
 
Problemlösung

Eine Aufstellung der überholten Tags und Attribute finden Sie im offiziellen HTML-Dokument unter HTML 4.01 Specification [http://www.w3.org/TR/html401/appendix/changes.html#h-A.3.1.2] .

Einige der folgenden Tags wurden in der Seite festgestellt:

  • APPLET
  • BASEFONT
  • CENTER
  • DIR
  • FONT
  • ISINDEX
  • MENU
  • S
  • STRIKE
  • U
  • LISTING
  • PLAINTEXT
  • XMP
 
 
Problemerklärung

überholte Tags und Attribute entsprechen nicht dem aktuellen HTML-Standard, Sie wurden vom W3C aus der Liste der Standards entfernt, da ihre Funktionen sich mit denen anderer Tags oder Technologien überschneiden und den aktuellen Entwicklungen und Trends von Standards zuwider laufen.

So ist beispielsweise das Tag FONT überholt, da sämtliche Eigenschaften von Schriftarten inzwischen mithilfe von CSS-Regeln ausgedrückt werden können (Redundanz). Außerdem wird so eine höhere Modularität und eine schärfere Trennung zwischen der Struktur und der Präsentation erreicht (Trends).
