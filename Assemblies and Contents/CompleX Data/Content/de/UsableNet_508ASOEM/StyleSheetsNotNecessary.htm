Seiten sollten ohne Stylesheets auskommen.  
Section 508 1194.22(d); WAI / WCAG 1.0 checkpoint 6.1
 
 
Problembeschreibung

Für die Seite werden Cascading Stylesheets (CSS) benötigt. Vergewissern Sie sich, dass das Verständnis und die Navigation der Seite auch ohne Stilinformationen gewährleistet ist.

 
 
Überprüfung

Vergewissern Sie sich, dass das Dokument auch ohne Stylesheets gelesen werden kann.

 
 
Problemerklärung

Cascading Stylesheets (CSS) stellen eine großartige Möglichkeit zur Trennung von Inhalt und Struktur von der Darstellung dar. Wenn Inhalte logisch angeordnet sind, werden sie normalerweise in sinnvoller Darstellung wiedergegeben, wenn Stylesheets deaktiviert sind oder nicht unterstützt werden. In manchen Situationen können CSS jedoch Probleme verursachen, die die Zugänglichkeit einer Webseite einschränken.

Ältere Browser bieten beispielsweise keine CSS-Unterstützung, und Webautoren sollten daher zur Präsentation von Inhalten oder für die Navigation keine Stile anwenden. Darüber hinaus sollten benutzerdefinierte Stilinformationen (wie Schriftgröße und -farbe) nicht durch auf der Seite definierte Stile beeinflusst werden.
