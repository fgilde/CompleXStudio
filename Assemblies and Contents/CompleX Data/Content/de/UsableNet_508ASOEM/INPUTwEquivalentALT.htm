INPUT mit äquivalentem ALT-Attribut  
Section 508 1194.22(a); WAI / WCAG 1.0 checkpoint 1.1
 
 
Problembeschreibung

Eine als Schaltfläche im Formular verwendete Grafik hat eine gültige Textbeschreibung.
Die Grafik sollte eine äquivalente Textbeschreibung besitzen.

 
 
Überprüfung

Testen Sie den Inhalt des ALT-Attributs für das INPUT-Element.
Die Beschreibung sollte die Auswirkungen erklären, die ein Klick auf die Schaltfläche hat.

Beachten Sie, dass beim Verwenden einer Maus zum Klicken auf die Grafik das Formular und die Koordinaten des Klicks an den Server übertragen werden. Wenn der Server je nach Position des Klicks unterschiedliche Aktionen durchführt, sind Benutzer von Textbrowsern im Nachteil. Daher sollten Sie folgende alternative Lösungen in Betracht ziehen:

  • Verwenden Sie einfache Schaltflächen zum Absenden (jeweils mit eigener Grafik) an Stelle einer einzigen grafischen Schaltfläche. Diese Schaltflächen können Sie über Stylesheets positionieren.
  • Verwenden Sie eine Client-Imagemap in Verbindung mit einem Skript.
 
 
Problemerklärung

Das ALT-Attribut beschreibt die entsprechende Grafik, damit auch Benutzer, die Grafiken im Browser deaktiviert haben, alle Navigationsoptionen der Seite verwenden können. Seit der Einführung von Browsern für PDAs und andere Handgeräte, die nur Text unterstützen, sind die ALT-Beschreibungen wichtiger denn je geworden.

Wenn für die Schaltflächengrafik keine äquivalente Textbeschreibung angegeben ist, haben Benutzer von Textbrowsern keine Ahnung, was beim Klicken auf die Schaltfläche geschieht.

Beachten Sie auch ein weiteres mögliches Problem. Beim Verwenden einer Maus zum Klicken auf die Grafik werden das Formular und die Koordinaten des Klicks an den Server übertragen. Wenn der Server je nach Position des Klicks unterschiedliche Aktionen durchführt, sind Benutzer von Textbrowsern im Nachteil. Daher sollten Sie folgende alternative Lösungen in Betracht ziehen:

  • Verwenden Sie einfache Schaltflächen zum Absenden (jeweils mit eigener Grafik) an Stelle einer einzigen grafischen Schaltfläche. Diese Schaltflächen können Sie über Stylesheets positionieren.
  • Verwenden Sie eine Client-Imagemap in Verbindung mit einem Skript.
