Neueste angemessene W3C-Technologien benutzen  
WAI / WCAG 1.0 Priorität 2 Checkpunkt 11.1
 
 
Problembeschreibung

Dies ist ein ganz allgemeiner Hinweis. Verwenden Sie am besten immer die neuesten von W3C freigegebenen Technologien, die der betreffenden Aufgabe angemessen ist.

 
 
Überprüfung

Diesbezügliche Empfehlungen des W3C finden Sie unter World Wide Web Consortium [http:w3.org] .

 
 
Problemerklärung

Die Programmierung einer Seite sollte immer nach dem aktuellen Stand der Technik durchgeführt werden, so dass die Browser (und andere Technologien) sich dann anpassen können. Auf diese Weise müssen die entwickelten Seiten nicht immer der Entwicklung der Browser folgen.

Dies bedeutet, dass Seiten beispielsweise im XHTML oder CSS erstellt werden, SMIL für das Einblenden von Beschriftungen in Multimediadaten verwendet wird, MathML für das Codieren mathematischer Formeln, XML und XSL zum vollständigen Trennen von Inhalten und Darstellung und P3P zur Darstellung von vertraulichen Daten.
