Relative Einheiten bei CSS-Boxeigenschaften verwenden  
WAI / WCAG 1.0 Priorität 2 Checkpunkt 3.4
 
 
Problembeschreibung

Die Seite bezieht sich auf CSS-Eigenschaften, die mithilfe von absoluten Einheiten (wie z. B. "in", "px" oder "mm") anstelle von relativen Einheiten (wie "%" oder "em") angegeben wurden. Diese Einheiten werden in den STYLE-Attributen oder in mit LINK verlinkten CSS-Dateien verwendet.

 
 
Überprüfung

Verwenden Sie relative statt absolute Einheiten für die Werte von Stylesheet-Eigenschaften. Ersetzen Sie bei allen Einheiten die in "px", "pt", "pc", "cm" oder "mm" angegebenen Werte durch Prozentangaben oder "em".

 
 
Problemerklärung

Nach Möglichkeit sollten immer relative Einheiten verwendet werden.

Auf diese Weise kann das optische Layout der Seite in so fern "fließend" gestaltet werden, als es sich dadurch besser an spezifische Browser-Einstellungen und diese unterstützende Anzeigen anpasst und ein Benutzer eine bestimmte Schriftgröße als Standard auswählen kann.
