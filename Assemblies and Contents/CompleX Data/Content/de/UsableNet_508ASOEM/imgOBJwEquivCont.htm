Grafik-OBJECT mit äquivalentem CONTENT  
Section 508 1194.22(a); WAI / WCAG 1.0 checkpoint 1.1
 
 
Problembeschreibung

Die OBJECT-Elemente mit eingebetteten Grafiken in diesem Dokument sollten eine entsprechende äquivalente Beschreibung zwischen den Tags <OBJECT> und </OBJECT> besitzen.

 
 
Überprüfung

Fügen Sie eine äquivalente aussagekräftige Beschreibung oder eine Grafik in den <OBJECT>-Block ein.

Die alternative Textbeschreibung sollte den Inhalt der Grafik und deren Funktion auf der Seite erklären.

 
 
Problemerklärung

Das OBJECT-Element bietet einen Mechanismus zum Definieren von alternativen Objektdarstellungen. In jeder Deklaration eines eingebetteten OBJECT-Elements können alternative Inhaltstypen angegeben werden. Wenn ein Browser das äußerste OBJECT nicht darstellen kann, wird stattdessen die Darstellung der Inhalte versucht, bei denen es sich z. B. um ein weiteres OBJECT-Element handeln kann.

Objekte zum Einbetten von Grafiken sollten über eine entsprechende äquivalente Beschreibung oder Grafik verfügen, sodass auch Benutzer mit nicht-grafikfähigen Browsern auf der Seite navigieren können.

Die Grafikbeschreibung sollte dieselben Informationen wie die Grafik selbst vermitteln und die Funktion der Grafik auf der Seite erklären: Warum die Grafik enthalten ist, welche Funktion übernommen wird und wie die Informationen ausgegeben werden.
