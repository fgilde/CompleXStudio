Verwendung von Spezial-Tags  
suggestions
 
 
Problembeschreibung

Die Seite enthält Spezial-Tags, die vom Webserver durch ein oder mehrere HTML-Tags ersetzt werden. Dadurch kann die Zugänglichkeit der ursprünglichen Datei nicht überprüft werden. Zur genauen Prüfung müsste die endgültige Version der Seite getestet werden. Sie könnten dazu die Seite in einem Browser öffnen, speichern und anschließend die gespeicherte Version testen.

 
 
Überprüfung

Rufen Sie über einen Browser die Seite vom Server ab. Die Seite, die im Browser geöffnet wird, sollte keine Spezial-Tags mehr enthalten. Speichern Sie die Seite im Browser (in Microsoft Internet Explorer unter "Datei" > "Speichern unter"), und testen Sie anschließend die Zugänglichkeit der gespeicherten Version.

 
 
Problemerklärung

Einige Webtechnologien, wie z. B. ColdFusion, verwenden spezialisierte Tags zur Erstellung bestimmter Informationen und Verhaltensweisen. Wenn die Seite beim Server angefordert wird, ersetzt dieser die spezialisierten Tags durch HTML-Code und sendet anschließend die überarbeitete Seite an den Browser. Die Seite, die der Benutzer empfängt, sollte keine Spezial-Tags mehr enthalten.

Die Zugänglichkeit einer Seite kann aus den folgenden Gründen möglicherweise nicht geprüft werden:

  • Die Site enthält möglicherweise ColdFusion- und ASP.NET-Seiten, die dynamisch erstellt werden.
  • Lokale Seiten sind möglicherweise Teil einer größeren Site und müssen daher im Gesamtzusammenhang getestet werden.
  • Lokale Seiten enthalten möglicherweise Spezial-Tags, die vor der Weitergabe der Seite an den Benutzerbrowser durch den Webserver ersetzt werden.
  • Bei Online-Seiten können außerdem tatsächliche Download-Zeiten und Server-Reaktionszeiten gemessen werden.
  • Unterschiedliche Browser erhalten möglicherweise leicht unterschiedliche Seiten vom Server.
  • Seiten enthalten möglicherweise Hyperlinks zu externen Sites.

Aus diesen Gründen kann die Zugänglichkeit einer Seite in der Originalversion nicht immer vollständig getestet werden. Sie müssen die Seite in der endgültigen Version testen.
