Eine Sammlung aller embed-Elemente, die im aktuellen Element enthalten sind. Mitglieder dieser Sammlung werden in der Quellcodereihenfolge sortiert. Internet Explorer unterstützt die Array-Schreibweise und runde Klammern zum Zugriff auf ein einzelnes Element in der Sammlung.

 
Objektmodellreferenz
  
document.embeds

var howMany = document.embeds.length;
