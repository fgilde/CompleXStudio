imagesNN 3 IE 4 DOM 1

Eine Sammlung (Array) aller img-Objekte, die im Dokument enthalten sind. Dieses Objekt wird nur in Browsern implementiert, die Bilder wie Objekte behandeln. Aus diesem Grund können Sie dieses Array-Objekt als einen konditionalen Schalter um Anweisungen verwenden, die Bilder austauschen oder im Voraus laden: 

if (document.images) {
    // process img element objects here
}
 

Internet Explorer 3.01 für den Macintosh bietet Unterstützung für Bilder als Objekte.

lengthNN 3 IE 4 DOM 1

Schreibgeschützt

Gibt die Anzahl von Elementen in der Sammlung zurück.
 
Beispiel
  
var howMany = document.images.length;
 
Wert

Eine ganze Zahl.

item( )NN 6 IE 4 DOM 1

item(index[, subindex]) item(index)

Gibt ein einzelnes Bildobjekt oder eine Objektsammlung zurück, das bzw. die dem mit dem Indexwert (oder bei IE optional mit dem index- und dem subindex-Wert) übereinstimmenden Element entspricht.

 
Parameter
 
  • Handelt es sich bei dem Parameter um eine auf Null basierende ganze Zahl (in Netscape 6 erforderlich), ist der zurückgegebene Wert ein einzelnes Element, das dem angegebenen nummerierten Objekt in der Quellcodereihenfolge entspricht (verschachtelt im aktuellen Element). Handelt es sich bei dem Parameter um eine Zeichenfolge, ist der zurückgegebene Wert eine Sammlung der Elemente, deren id- oder name-Eigenschaft mit dieser Zeichenfolge übereinstimmt.
  • Wenn Sie eine Zeichenfolge als ersten Parameter angeben (nur IE), können Sie den zweiten Parameter verwenden, um eine auf Null basierende ganze Zahl anzugeben, die ein bestimmtes Element aus der Sammlung abruft, dessen Eigenschaft id oder name mit der Zeichenfolge des ersten Parameters übereinstimmt.
 
Zurückgegebener Wert

Ein Objekt oder eine Objektsammlung (Array). Falls keine Übereinstimmung mit den Parametern vorliegt, lautet der zurückgegebene Wert null.

namedItem( )NN 6 IE 6 DOM 2

namedItem(IDOrName)

Gibt ein einzelnes Objekt zurück, das dem mit dem Wert der Parameterzeichenfolge übereinstimmenden Element entspricht.

 
Parameter
 
  • Die Zeichenfolge, die denselben Wert wie das id- oder name-Attribut des gewünschten Elements enthält.
 
Zurückgegebener Wert

Eine Objektreferenz. Falls keine Übereinstimmung mit den Parametern vorliegt, lautet der zurückgegebene Wert null.

tags( )NN nicht zutreffend IE 4 DOM nicht zutreffend

tags(tagName)

Gibt eine Sammlung von Objekten zurück (aus allen in der aktuellen Objektsammlung enthaltenen Objekten), deren Tags mit dem tagName-Parameter übereinstimmen. In diesem Fall überflüssig, da alle Elemente dasselbe img-Tag aufweisen.

 
Parameter
 
  • Eine Zeichenfolge, die das Element-Tag enthält, z. B. document.images.tags("img").
 
Zurückgegebener Wert

Eine Objektsammlung (Array). Falls keine Übereinstimmung mit den Parametern vorliegt, ist der zurückgegebene Wert ein Array mit der Länge Null.

urns( )NN nicht zutreffend IE 5(Win) DOM nicht zutreffend

urns(URN)

Gibt eine Sammlung verschachtelter Elementobjekte zurück, denen Verhalten zugewiesen sind und deren URNs mit dem URN-Parameter übereinstimmen.

 
Parameter
 
  • Eine Zeichenfolge mit dem URN einer lokalen oder externen Verhaltensdatei.
 
Zurückgegebener Wert

Eine Objektsammlung (Array). Falls keine Übereinstimmung mit den Parametern vorliegt, ist der zurückgegebene Wert ein Array mit der Länge Null.
