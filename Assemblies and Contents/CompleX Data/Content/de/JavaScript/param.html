paramNN 6 IE 5 DOM 1

Das param-Objekt spiegelt das param-Element wider, das Variablenwerte an ActiveX-Objekte (nur IE/Windows), Java-Applets und bestimmte Plug-Ins übergibt. Solche Programme werden geschrieben, um während der Initialisierung Namens-/Wertpaare von Parametern zu lesen, damit die entsprechenden Werte (z. B. die URL einer Sounddatei) beim Programmstart zur Verfügung stehen. IE für Windows weist einigen ActiveX-Steuerelementen im Allgemeinen eine ganze Reihe von Parametern zu, wenngleich möglicherweise nur einige von ihnen explizit im Quellcode definiert sind. Obwohl Eigenschaften gelesen und geschrieben werden können, werden nach dem Laden der Seite zugeordnete Werte nicht an das externe Programm weitergeleitet.

nameNN 6 IE 5 DOM 1

Lesen/Schreiben

Dies ist der Name des Parameters des externen Programms, auf den sich ein Wert im param-Element bezieht.

 
Beispiel
  
var pName = document.getElementById("audioParam2").name;
 
Wert

Zeichenfolge.
 
Standardwert

Keiner.

typeNN 6 IE 5 DOM 1

Lesen/Schreiben

Stellt den MIME-Typ für ein param-Element bereit, dessen valuetype-Attribut auf "ref" gesetzt ist.

 
Beispiel
  
if (document.getElementById("myParam").valueType == "ref") {
    var pType = document.getElementById("myParam").type;
}
 
Wert

MIME-Typ in Form einer Zeichenfolge (Groß-/Kleinschreibung unwichtig). Einen Katalog der registrierten MIME-Typen finden Sie unter ftp://ftp.isi.edu/in-notes/iana/assignments/media-types/.

 
Standardwert

Keiner.

valueTypeNN 6 IE 5(Mac)/6(Win) DOM 1

Lesen/Schreiben

Gibt die Zeichenfolgenklassifikation des vom Element gesetzten Parameters an.

 
Beispiel
  
var pValType = document.getElementById("volumeParam").valueType;
 
Wert

Zeichenfolgenkonstante: data | object | ref.

 
Standardwert

data
