Wird ausgelöst, nachdem der Benutzer (über das Menü Bearbeiten, eine Tastenkombination oder ein Kontextmenü) den Befehl Kopieren aufgerufen hat, um den ausgewählten Inhalt in die Zwischenablage zu kopieren. Mithilfe einer Ereignisbehandlungsroutinenfunktion für dieses Ereignis können Sie den Kopiervorgang erweitern und beispielsweise zusätzliche Daten Ihrer Wahl in das clipboardData-Objekt übernehmen (die sich mit der Ereignisbehandlungsroutine onpaste lesen und je nach Bedarf einsetzen lassen).

 

Wenn Sie dem Benutzer den Zugriff auf einen Kopieren-Menübefehl für ein ansonsten nicht bearbeitbares Element ermöglichen möchten, stellen Sie in der Ereignisbehandlungsroutine onbeforecopy desselben Objekts, für das Sie auch die Ereignisbehandlungsroutine oncopy definiert haben, die Eigenschaft event.returnValue auf false ein. Möchten Sie hingegen verhindern, dass der Benutzer Body-Inhalte kopiert, setzen Sie die Eigenschaft event.returnValue in der Ereignisbehandlungsroutine oncopy auf false. Denken Sie jedoch daran, dass diese Taktik keinen zuverlässigen Schutz vor Benutzern bietet, die Ihre geschätzten Inhalte gezielt zu kopieren versuchen.

 
Typische Ziele

Dargestellte Elemente mit Ausnahme von Formularsteuerelementen.
