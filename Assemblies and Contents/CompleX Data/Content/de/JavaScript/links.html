linksNN 2 IE 3 DOM 1

Eine Sammlung aller a- und area-Elemente, die sich aufgrund der ihnen zugewiesenen href-Attribute wie Hyperlinks (statt nur wie Anker) verhalten. Mitglieder dieser Sammlung werden in der Quellcodereihenfolge sortiert. Navigator und Internet Explorer unterstützen die Array-Schreibweise zum Zugriff auf einen einzelnen Hyperlink in der Sammlung (z. B. document.links[0] oder document.links["section3"]). Falls der Name des Hyperlink als Indexwert verwendet werden soll (immer als Bezeichnerstring), müssen Sie den Wert des name-Attributs anstelle des id-Attributs benutzen. Zur Verwendung des id-Attributs in einer Referenz auf einen Anker greifen Sie über eine document.all.elementID-Referenz (nur IE) oder eine document.getElementByID("elementID")-Referenz auf das Objekt zu.

Beispiel
  
var howMany = document.links.length;
 
Wert

Eine ganze Zahl.

item( )NN 6 IE 4 DOM 1

item(index) item(index[, subindex])

Gibt ein einzelnes Objekt oder eine Objektsammlung zurück, das bzw. die dem mit dem Indexwert (oder bei IE/Windows optional mit dem Index- und dem Subindexwert) übereinstimmenden Element entspricht.

 
Parameter
 
  • Handelt es sich bei dem Parameter um eine auf Null basierende ganze Zahl (wie für Netscape 6 erforderlich), ist der zurückgegebene Wert ein einzelnes Element, das dem angegebenen nummerierten Objekt in der Quellcodereihenfolge (verschachtelt im aktuellen Element) entspricht. Handelt es sich bei dem Parameter um eine Zeichenfolge (wie für IE/Windows zulässig), ist der zurückgegebene Wert eine Sammlung der Elemente, deren Eigenschaft id oder name mit der angegebenen Zeichenfolge übereinstimmt.
  • Wenn Sie eine Zeichenfolge als ersten Parameter angeben (in IE/Windows), können Sie den zweiten Parameter verwenden, um eine auf Null basierende ganze Zahl anzugeben, die ein bestimmtes Element aus der Sammlung abruft, dessen Eigenschaft id oder name mit der Zeichenfolge des ersten Parameters übereinstimmt.
 
Zurückgegebener Wert

Ein Objekt oder eine Objektsammlung (Array). Falls keine Übereinstimmung mit den Parametern vorliegt, lautet der zurückgegebene Wert null.

namedItem( )NN 6 IE 6 DOM 1

namedItem(IDOrName)

Gibt ein einzelnes Objekt (in Netscape 6) oder eine Objektsammlung zurück, das bzw. die dem mit dem Wert der Parameterzeichenfolge übereinstimmenden Element entspricht.

 
Parameter
 
  • Die Zeichenfolge, die denselben Wert wie das id- oder name-Attribut des gewünschten Elements enthält.
 
Zurückgegebener Wert

Ein Objekt (in Netscape 6) oder eine Objektsammlung (Array). Falls keine Übereinstimmung mit den Parametern vorliegt, lautet der zurückgegebene Wert null.

tags( )NN nicht zutreffend IE 4 DOM nicht zutreffend

tags(tagName)

Gibt eine Sammlung von Objekten zurück (aus allen in der aktuellen Sammlung enthaltenen Objekten), deren Tags mit dem tagName-Parameter übereinstimmen. Ermöglicht die Unterscheidung zwischen den Sammlungen von a- und area-Elementen.

 
Parameter
 
  • Eine Zeichenfolge, die das Element-Tag enthält, z. B. document.links.tags("a").
 
Zurückgegebener Wert

Eine Objektsammlung (Array). Falls keine Übereinstimmung mit den Parametern vorliegt, ist der zurückgegebene Wert ein Array mit der Länge Null.

urns(URN)

Gibt eine Sammlung verschachtelter Elementobjekte zurück, denen Verhalten zugewiesen sind und deren URNs mit dem URN-Parameter übereinstimmen.

 
Parameter
 
  • Eine Zeichenfolge mit dem URN einer lokalen oder externen Verhaltensdatei.
 
Zurückgegebener Wert

Eine Objektsammlung (Array). Falls keine Übereinstimmung mit den Parametern vorliegt, ist der zurückgegebene Wert ein Array mit der Länge Null.
