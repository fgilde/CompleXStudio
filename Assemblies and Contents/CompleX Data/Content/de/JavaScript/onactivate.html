Aufwärtsverbreitung: Ja; Abbruch möglich: Nein

Wird ausgelöst, wenn ein Objekt zum aktiven Objekt wird. Ein Objekt, das den Fokus erhält, wird automatisch aktiv, ein dargestelltes Element kann jedoch auch das aktive Element sein, wenn es nicht den Fokus hat. Es kann jeweils immer nur ein Element aktiv sein. Weitere Informationen finden Sie in der Abhandlung der Methode setActive( ) im Abschnitt über die gemeinsamen Objekte in Kapitel 9. Wenn ein Element den Fokus erhalten hat, wird das onactivate-Ereignis vor dem onfocus-Ereignis ausgelöst.

 
Typische Ziele

Alle dargestellten Elemente sowie die Objekte document und window.
