Aufwärtsverbreitung: Ja; Abbruch möglich: Ja

Wird ausgelöst, unmittelbar bevor ein vom Benutzer aufgerufener Kopieren-Befehl (über das Menü Bearbeiten, eine Tastenkombination oder ein Kontextmenü) das Verschieben des ausgewählten Inhalts in die Zwischenablage des Systems abgeschlossen hat. An dieser Stelle des Kopiervorgangs kann eine von dieser Ereignisbehandlungsroutine aufgerufene Funktion Verarbeitungsschritte ausführen, die die normale Kopieren-Aktion des Systems ergänzen oder ersetzen. Zum Beispiel ist es möglich, zusätzliche Informationen aus einem Element (etwa zum effektiven Stil des Elements, das einen ausgewählten Text enthält) im IE-Objekt clipboardData (siehe Kapitel 9) abzulegen, damit sie später mithilfe der Ereignisbehandlungsroutine onbeforepaste weiterverarbeitet werden können. Ein Abbruch des onbeforecopy-Ereignisses verhindert nicht, dass der Benutzer eine Auswahl kopiert.

 
Typische Ziele

Dargestellte Elemente mit Ausnahme von Formularsteuerelementen.
