anchorsNN 2 IE 3 DOM 1

Eine Sammlung aller a-Elemente, die sich aufgrund der ihnen zugewiesenen name-Attribute wie Anker (und nicht wie Hyperlinks) verhalten. Mitglieder dieser Sammlung werden in der Quellcodereihenfolge sortiert. Navigator und Internet Explorer unterstützen eine Array-Schreibweise zum Zugriff auf einen einzelnen Anker in der Sammlung (z. B. document.anchors[0], document.anchors["section3"]). In Internet Explorer 4 kann der Indexwert darüber hinaus in runde (statt eckige) Klammern gesetzt werden (z. B. document.anchors(0)). Falls der Name des Ankers als ein Indexwert verwendet werden soll (immer als Stringbezeichner), müssen Sie darauf achten, den Wert des name-Attributs anstelle des id-Attributs zu verwenden. Zur Verwendung des id-Attributs in einer Referenz auf einen Anker greifen Sie über eine document.all.elementID-Referenz (nur IE) oder eine document.getElementByID("elementID")-Referenz auf das Objekt zu.

lengthNN 2 IE 3 DOM 1

Schreibgeschützt

Gibt die Anzahl von Elementen in der Sammlung zurück.
 
Beispiel
  
var howMany = document.anchors.length;
 
Wert

Eine ganze Zahl.

item( )NN 6 IE 4 DOM 1

item(index[, subindex]) item(index)

Gibt ein einzelnes Ankerobjekt oder eine Ankerobjektsammlung zurück, das bzw. die dem mit dem Indexwert (oder bei IE optional mit dem index- und dem subindex-Wert) übereinstimmenden Element entspricht.

 
Parameter
 
  • Handelt es sich bei dem Parameter um eine auf Null basierende ganze Zahl (erforderlich in Netscape 6), ist der zurückgegebene Wert ein einzelnes Element, das dem angegebenen Element in der Quellcodereihenfolge (verschachtelt im aktuellen Element) entspricht. Handelt es sich bei dem Parameter um eine Zeichenfolge, ist der zurückgegebene Wert eine Sammlung der Elemente, deren id- oder name-Eigenschaft mit dieser Zeichenfolge übereinstimmt.
  • Wenn Sie eine Zeichenfolge als ersten Parameter angeben (nur IE), können Sie den zweiten Parameter verwenden, um einen auf Null basierenden Index anzugeben, der das angegebene Element aus der Sammlung abruft, dessen id- oder name-Eigenschaft mit der Zeichenfolge des ersten Parameters übereinstimmt.
 
Zurückgegebener Wert

Ein Ankerobjekt oder eine Ankerobjektsammlung (Array). Falls keine Übereinstimmung mit den Parametern vorliegt, lautet der zurückgegebene Wert null.

namedItem( )NN 6 IE 6 DOM 1

namedItem(IDOrName)

Gibt ein einzelnes Ankerobjekt oder eine Ankerobjektsammlung zurück, das bzw. die dem mit dem Wert der Parameterzeichenfolge übereinstimmenden Element entspricht.

 
Parameter
 
  • Die Zeichenfolge, die denselben Wert wie das id- oder name-Attribut des gewünschten Elements enthält.
 
Zurückgegebener Wert

Ein Ankerobjekt oder eine Ankerobjektsammlung (Array). Falls keine Übereinstimmung mit den Parametern vorliegt, lautet der zurückgegebene Wert null.
