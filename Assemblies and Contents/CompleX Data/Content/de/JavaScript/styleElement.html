Das style-Elementobjekt spiegelt das HTML-Element style wider. Dieses Objekt unterscheidet sich vom style-Objekt, auf das als Eigenschaft praktisch eines jeden Elements in einem Dokument zugegriffen wird. Das Elementobjekt style wird in einem Dokument über das <style>-Tag erstellt, dem eine eindeutige ID zugewiesen sein kann. Das Eigenschaftsobjekt style hingegen enthält alle Stileigenschaften und deren aktuellen Werte, die für ein bestimmtes Element definiert wurden.

disabledNN 6 IE 4 DOM 1

Lesen/Schreiben

Legt fest, ob Regeln im Stylesheet auf die ausgewählten Elemente angewendet werden sollen. Obwohl das entsprechende disabled-Attribut in Internet Explorer 4 nicht funktioniert, wird das gesamte Stylesheet deaktiviert, wenn die Eigenschaft disabled auf true gesetzt wird. Sie können während des Erstellens einer Seite eine Schaltfläche zum Aktivieren/Deaktivieren von Stylesheets integrieren, um zu überprüfen, ob die Seite in allen Browsern gleich aussieht.

 
Beispiel
  
document.getElementById("mainStyle").disabled = true;
 
Wert

Boolescher Wert: true | false.
 
Standardwert

false
mediaNN 6 IE 4 DOM 1

Lesen/Schreiben

Gibt das beabsichtige Ausgabegerät für die Regeln des style-Elements an. Mit dem Attribut media können Browser in der Zukunft Inhalte an bestimmte Gerätetypen, wie z. B. Handheld-Computer, sprachfähige Digitalisiergeräte oder Fernsehgeräte, anpassen.

 
Beispiel
  
document.getElementById("myStyle").media = "print";
 
Wert

Einer der folgenden Konstantenwerte in Form einer Zeichenfolge: all | print | screen.

 
Standardwert

all

Schreibgeschützt

Gibt ein styleSheet-Objekt (W3C-DOM-Typ CSSStyleSheet) zurück, das das durch das style-Element definierte Stylesheet darstellt. Hierbei handelt es sich um ein alternatives (nicht standardgemäßes) Verfahren zum Referenzieren eines styleSheet-Objekts. Es wird empfohlen, stattdessen die document.styleSheets-Sammlung zu verwenden.

 
Beispiel
  
var oneSheet = document.getElementById("myStyle").sheet;
 
Wert

Referenz auf ein styleSheet-Objekt (W3C-DOM-Typ CSSStyleSheet).

 
Standardwert

Keiner.
