Wenn die Eigenschaft oder Methode eines ActiveX-Steuerelements eine Wertesammlung zurückgibt, funktioniert die übliche JavaScript-Verarbeitung von Sammlungen (bei der sie als Arrays behandelt werden) für solche Werte nicht. Das Enumerator-Objekt steuert einen Zeiger in der Sammlungsliste, mit dessen Hilfe JavaScript in der Lage ist, die Elemente einer solchen Sammlung zu referenzieren. Weitere Informationen hierzu finden Sie unter http://msdn.microsoft.com/library/default.asp?url=/library/en-us/script56/html/js56jsobjenumerator.asp.
