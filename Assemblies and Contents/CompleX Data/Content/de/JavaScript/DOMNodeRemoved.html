DOMNodeRemovedNN 6 IE –- DOM 2

Aufwärtsverbreitung: Ja; Abbruch möglich: Nein

Wird in einer Node ausgelöst, wenn sie explizit aus einem Nodecontainer entfernt wird. Wenn Sie dem Container einer entfernten Node einen Event-Listener für dieses Ereignis zuweisen, wird das DOMNodeRemoved-Ereignis in der entfernten Node (also im srcElement des event-Objekts) ausgelöst. Wenn sich das Ereignis anschließend nach oben (d. h. von der ursprüngliche Position der Node fort) verbreitet, kann die Event-Listener-Funktion immer noch den alten Container ermitteln, indem sie die Ereigniseigenschaft relatedNode liest.

 
Typische Ziele

Alle dargestellten Nodes.
