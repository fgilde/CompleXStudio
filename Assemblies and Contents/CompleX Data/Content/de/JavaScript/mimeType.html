Das mimeType-Objekt gehört zum navigator-Objekt. Das Objekt stellt eine MIME-Typspezifikation dar. Mit den Eigenschaften dieses Objekts können Skripts feststellen, ob der Browser für die Verarbeitung externer Inhalte eines bestimmten MIME-Typs geeignet ist, bevor diese Inhalte vom Server geladen werden. Diese Eigenschaften werden in einem internen Dokument widergespiegelt, das angezeigt wird, wenn Sie im Navigator die Menüoption Info über Plug-ins wählen. Internet Explorer 5 für den Macintosh implementiert dieses Schema, Explorer für Windows hingegen verwendet ein vollkommen anderes System, um festzustellen, ob ein externes Medium über das object-Element unterstützt wird.

enabledPluginNN 3 IE 5(Mac) DOM nicht zutreffend

Schreibgeschützt

Gibt eine plugin-Objektreferenz zurück, die dem Plug-In entspricht, das derzeit für die Wiedergabe aller eingehenden Daten des aktuellen MIME-Typs eingestellt ist. Die Eigenschaften des zurückgegebenen plugin-Objekts können dann genauer untersucht werden, z. B. um seinen Namen abzurufen.

 
Beispiel
  
var plugName = navigator.mimeTypes["video/mpeg"].enabledPlugin.name;

suffixesNN 3 IE 5(Mac) DOM nicht zutreffend

Schreibgeschützt

Gibt eine durch Kommas getrennte Zeichenfolgenliste der Dateinamenserweiterungen zurück, die mit dem mimeType-Objekt, wie es von dem für diesen MIME-Typ aktivierten Plug-In unterstützt wird, verknüpft sind. Das QuickTime-Plug-In beispielsweise erkennt die folgenden Dateinamenserweiterungen für Dateien vom Typ video/avi: 

avi, vfw
 

Wenn Sie alle im Browser registrierten mimeType-Objekte durchlaufen, um nach einer Übereinstimmung für eine bestimmte Dateinamenserweiterung zu suchen, können Sie anschließend überprüfen, ob für das übereinstimmende mimeType-Objekt ein Plug-In installiert ist (mit Hilfe der enabledPlugin-Eigenschaft).

 
Beispiel
  
var suff = navigator.mimeTypes["audio/mpeg"].suffixes;

Schreibgeschützt

Gibt eine Zeichenfolgenversion des MIME-Typs zurück, der mit dem mimeType-Objekt verknüpft ist. Sie können z. B. alle mimeType-Objekte durchlaufen, um nach einem Objekt zu suchen, das einem bestimmten MIME-Typ entspricht (application/x-midi). Das gefundene mimeType-Objekt kann dann weiter untersucht werden, um festzustellen, ob es derzeit unterstützt wird und aktiviert ist.

 
Beispiel
  
var MType = navigator.mimeTypes[3].type;
