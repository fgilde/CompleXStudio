Eine Sammlung (Array) mit styleSheet-Objekten, die mittels der @import-Regel in ein explizites styleSheet-Objekt importiert wurden. Mit anderen Worten, ein styleSheet-Objekt, das eine @import-Regel verwendet, behandelt dieses importierte Stylesheet als verschachteltes Objekt, das seinerseits jedoch ein vollwertiges styleSheet-Objekt darstellt. Dies ermöglicht es Ihnen, auf die rule-Objekte innerhalb der einzelnen importierten Stylesheets zuzugreifen. Der Code im folgenden Beispiel durchsucht alle styleSheet-Objekte des Dokuments der Reihe nach nach importierten Stylesheets: 

for (var i = 0; i< document.styleSheets.length; i++) {
    for (var j = 0; j < document.styleSheets[i].imports.length; j++) {
        // process each imported style sheet, referenced
        // here as document.styleSheets[i].imports[j]
    }
}

Gibt ein einzelnes importiertes styleSheet-Objekt zurück, das mit dem Indexwert in der Quellcodereihenfolge der @import-Regeln übereinstimmt. IE 5 für Macintosh gibt fälschlicherweise den Wert null zurück.
