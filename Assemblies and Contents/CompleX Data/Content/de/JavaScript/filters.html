Stellt eine Sammlung aller Filter bereit, die dem aktuellen Element zugeordnet sind. Internet Explorer unterstützt die Array-Schreibweise und runde Klammern zum Zugriff auf ein einzelnes Element in der Sammlung. Filter sind in IE für Macintosh bis einschließlich 5.1 nicht verfügbar.

 
Objektmodellreferenz
  
document.getElementById("elementID").filters
 
Objektspezifische Eigenschaften
 
length
 
Objektspezifische Methoden
 
item( )namedItem( )
lengthNN nicht zutreffend IE 4(Win) DOM nicht zutreffend

Schreibgeschützt

Gibt die Anzahl von Elementen in der Sammlung zurück.
 
Beispiel
  
var howMany = document.body.filters.length;
 
Wert

Eine ganze Zahl.

item( )NN nicht zutreffend IE 4(Win) DOM nicht zutreffend

item(index)

Gibt ein Filterobjekt mit dem Filter zurück, der dem Indexwert in der Quellcodereihenfolge entspricht.

 
Parameter
 
  • Eine auf Null basierende ganze Zahl, die dem angegebenen Element in der Quellcodereihenfolge entspricht.
 
Zurückgegebener Wert

Referenz auf ein Filterobjekt. Falls keine Übereinstimmung mit den Parametern vorliegt, lautet der zurückgegebene Wert null.

namedItem( )NN nicht zutreffend IE 6(Win) DOM nicht zutreffend

namedItem(IDOrName)

Gibt ein einzelnes Filterobjekt oder eine Objektsammlung mit Filtern zurück, die mit dem Wert der Parameterzeichenfolge übereinstimmen.

 
Parameter
 
  • Die Zeichenfolge, deren Wert mit dem Namen des gewünschten Filters übereinstimmt.
 
Zurückgegebener Wert

Ein Filterobjekt oder eine Objektsammlung (Array) mit Filterobjekten. Falls keine Übereinstimmung mit den Parametern vorliegt, lautet der zurückgegebene Wert null.
