Wird ausgelöst, nachdem ein Skript ein Elementattribut hinzugefügt, entfernt oder seinen Wert geändert hat. Das Hinzufügen oder Entfernen von Attributen geschieht ausschließlich mithilfe der Methode setAttribute( ) bzw. removeAttribute( ), der Wert eines vorhandenen Attributs hingegen kann entweder mithilfe der Methode setAttribute( ) oder durch Zuweisen eines neuen Werts an die einem Attribut entsprechende Eigenschaft eines Elementobjekts geändert werden. Einer Reihe von Ereignisobjekteigenschaften können Detailinformationen zu diesem Ereignis entnommen werden. Weitere Informationen finden Sie unter den folgenden Eigenschaften des event-Objekts in Kapitel 9: attrName, attrChange, prevValue, newValue und relatedNode.

 
Typische Ziele

Alle dargestellten Elemente.
