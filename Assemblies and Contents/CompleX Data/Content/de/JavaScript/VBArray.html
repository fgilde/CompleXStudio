Das VBArray-Objekt ermöglicht es JavaScript, mit geschützten Visual Basic-Arrays zu kommunizieren. Diese Arrays sind schreibgeschützt, können mehrdimensional sein und werden manchmal von ActiveX-Steuerelementen als Wert zurückgegeben. Über die Methoden dieses Objekts erhält JavaScript Zugriff auf die Daten in einem VBArray. Weitere Informationen hierzu finden Sie unter http://msdn.microsoft.com/scripting/jscript/doc/jsobjVBArray.htm.
