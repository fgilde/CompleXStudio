Lesen/Schreiben

Gibt das dem Element zugeordnete style-Objekt an, das durch das explizite Zuweisen von Werten zum style-Attribut des Elements im Tag festgelegt wird. In der W3C-DOM-Objektterminologie wird dieses Objekt präziser als CSSStyleDeclaration-Objekt bezeichnet. Im Hinblick auf die Skripterstellung mit Stylesheets können Sie die style-Objekte von IE und Navigator in der Regel jedoch in ganz ähnlicher Weise behandeln. Diese Eigenschaft dient als Gateway zum Lesen und Schreiben individueller Stylesheet-Eigenschaftseinstellungen für ein Element. Informationen zum Lesen der effektiven Stylesheet-Eigenschaften eines Elements (einschließlich importierter Stylesheet-Attribute) finden Sie in den Abhandlungen der Eigenschaft currentStyle (für IE) und der Methode window.getComputedStyle( ) (für Netscape 6) weiter vorne in diesem Kapitel.

 
Beispiel
  
document.getElementById("elementID").style.fontSize = "14pt";
