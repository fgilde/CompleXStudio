ActiveXObjectNN nicht zutreffend IE 4(Win) ECMA nicht zutreffend

Internet Explorer für Windows stellt ein direktes Portal zwischen einer Webseite und einem beim Windows-System bereits registrierten ActiveX-Steuerelement (oder, um den Windows-Jargon zu verwenden, einem Automatisierungsobjekt) bereit. Indem Sie eine Instanz des ActiveXObject-Objekts erstellen, erzeugen Sie eine Referenz auf dieses Steuerelement, mit deren Hilfe Sie in Ihren Skripts auf dessen Eigenschaften zugreifen oder dessen Methoden aufrufen können. Hilfestellung bei der Offenlegung der Methoden und Eigenschaften eines Automatisierungsobjekts finden Sie auf der Microsoft Developer Network-Website (http://msdn.microsoft.com). Eine Microsoft-Utility namens OLE/COM Object Viewer kann dem engagierten Entwickler ebenfalls gute Dienste leisten. Informationen zum Einstieg in die Arbeit mit diesem Dienstprogramm finden Sie unter http://msdn.microsoft.com/scripting/jscript/doc/jsobjActiveXObject.htm. Informationen zum Abrufen einer Referenz auf ein Automatisierungsobjekt über seinen lokalen Pfadnamen finden Sie in der Abhandlung der Methode GetObject( ).
