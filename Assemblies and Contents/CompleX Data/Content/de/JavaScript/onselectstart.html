Aufwärtsverbreitung: Ja; Abbruch möglich: Ja

Wird ausgelöst, unmittelbar nachdem der Benutzer damit beginnt, eine Auswahl auf ein Body-Element oder den Text eines Formularsteuerelements zu ziehen. Wenn die Auswahl mehrere Elemente umfasst, wird nur ein Ereignis ausgelöst, dessen Ziel das erste Element in der Auswahl ist. Durch Abbrechen dieses Ereignisses im <body>-Tag (onselectstart="return false") können Sie eine unerwünschte Auswahl durch den Benutzer oder versehentliche Bildlaufinteraktionen unterbinden.

 
Typische Ziele

Alle dargestellten Elemente.
