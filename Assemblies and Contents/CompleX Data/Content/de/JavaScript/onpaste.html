Wird ausgelöst, nachdem der Benutzer (über das Menü Bearbeiten, eine Tastenkombination oder ein Kontextmenü) den Befehl Einfügen aufgerufen hat, um eine Kopie des Inhalts der Zwischenablage in den ausgewählten Inhalt einzufügen. Mithilfe einer Ereignisbehandlungsroutinenfunktion für dieses Ereignis können Sie den Einfügen-Vorgang erweitern und beispielsweise zusätzliche Daten aus dem clipboardData-Objekt abrufen (die von einer der Ereignisbehandlungsroutinenfunktionen im Rahmen des Kopier- oder Ausschneiden-Vorgangs dort abgelegt wurden).

 

Wenn Sie dem Benutzer den Zugriff auf einen Einfügen-Menübefehl für ein ansonsten nicht bearbeitbares Element ermöglichen möchten, stellen Sie in der Ereignisbehandlungsroutine onbeforepaste desselben Objekts, für das Sie auch die Ereignisbehandlungsroutine onpaste definiert haben, die Eigenschaft event.returnValue auf false ein. Möchten Sie hingegen verhindern, dass der Benutzer Inhalte aus der Zwischenablage des Systems einfügt, setzen Sie die Eigenschaft event.returnValue in der Ereignisbehandlungsroutine onpaste auf false.

 
Typische Ziele

Alle dargestellten Elemente.
