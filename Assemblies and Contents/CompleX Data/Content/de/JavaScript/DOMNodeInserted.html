DOMNodeInsertedNN 6 IE –- DOM 2

Aufwärtsverbreitung: Ja; Abbruch möglich: Nein

Wird in einer Node ausgelöst, wenn sie explizit in einen vorhandenen Nodecontainer eingefügt wird. Wenn Sie dem Empfänger einer eingefügten (oder angehängten) Node einen Event-Listener für dieses Ereignis zuweisen, wird das DOMNodeInserted-Ereignis in der eingefügten Node (also im srcElement des event-Objekts) ausgelöst. Wenn sich das Ereignis anschließend nach oben (d. h. von der Einfügeposition fort) verbreitet, kann die Event-Listener-Funktion immer noch den neuen Container ermitteln, indem sie die Ereigniseigenschaft relatedNode liest. Beachten Sie, dass beim Entfernen einer Node, die von einer anderen Position im selben Dokument übernommen wird, in dieser Node das DOMNodeRemoved-Ereignis ausgelöst wird, bevor sie ihren ursprünglichen Container verlässt (sodass sich das Ereignis nach oben an seinen ursprünglichen Container verbreiten kann).

 
Typische Ziele

Alle dargestellten Nodes.
