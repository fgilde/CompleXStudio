compactNN 6 IE 4 DOM 1

Lesen/Schreiben

Wenn sie auf true gesetzt ist, weist die Eigenschaft compact den Browser an, ein verwandtes dt- und dd-Paar in derselben Zeile darzustellen, sofern der zur Verfügung stehende Platz dies erlaubt. Dieser kompakte Stil wurde für dt-Elemente konzipiert, die nur aus einigen wenigen Zeichen bestehen.

 
Beispiel
  
document.getElementById("maindl").compact = true;
 
Wert

Boolescher Wert: true | false.
 
Standardwert

false

documentNN nicht zutreffend IE 4 DOM nicht zutreffend

Schreibgeschützt

Gibt eine Referenz auf das document-Objekt zurück, das das aktuelle Element enthält. Potenziell hilfreich in Funktionen zur Verarbeitung von Objektreferenzen, die aus Ereigniseigenschaften abgerufen oder als ID-Zeichenfolgen übergeben wurden. Die entsprechende W3C-DOM-Eigenschaft ist ownerDocument.

 
Beispiel
  
var currDoc = document.getElementById("elementID").document;
 
Wert

document-Objektreferenz.
 
Standardwert

Das aktuelle document-Objekt.
