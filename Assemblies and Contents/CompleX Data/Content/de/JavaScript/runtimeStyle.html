runtimeStyleNN nicht zutreffend IE 5(Win) DOM nicht zutreffend

Das runtimeStyle-Objekt (eine Eigenschaft aller HTML-Elementobjekte in IE für Windows ab Version 5) funktioniert wie ein übermächtiges style-Objekt. Seine Eigenschaftseinstellungen werden vorrangig vor allen in explizit kodierten Stildefinitionen vorgenommenen Einstellungen der entsprechenden Eigenschaften verarbeitet. Auf diese Weise setzt es alle globalen, importierten, verknüpften und im Code vorgenommenen Stildefinitionen außer Kraft. Dieses Objekt enthält nahezu dieselbe lange Liste von Eigenschaften und Methoden wie das style-Objekt.
