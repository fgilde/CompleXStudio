onsubmitNN 2 IE 3 DOM 2

Aufwärtsverbreitung: Nein (IE); Ja (NN 6); Abbruch möglich: Ja

Wird ausgelöst, wenn ein form-Element von einem Submit-input-Element eine Anforderung zum Übertragen des Formulars erhält (nicht jedoch, wenn die Anforderung von einer submit( )-Methode im Skript stammt). Mit diesem Ereignis kann vor der Übertragung des Formulars an den Server eine letzte clientseitige Formularvalidierung eingeleitet werden. Wenn Sie das Ereignis abbrechen, unterbindet ein Skript in der onsubmit-Ereignisbehandlungsroutine die Übertragung.

 
Typische Ziele

Das form-Element.
