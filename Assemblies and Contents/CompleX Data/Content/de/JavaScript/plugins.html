Navigator und Internet Explorer verfügen beide über ein plugins-Array. Es handelt sich dabei jedoch um recht unterschiedliche Objektsammlungen. Das plugins-Array von Navigator und IE/Macintosh ist eine Eigenschaft des navigator-Objekts. Jedes Element im navigator.plugins-Array stellt ein im Browser installiertes Plug-In dar (beim letzten Start des Browsers registriert). Weitere Informationen finden Sie unter dem plugin-Objekt.

 

Auf der Windows-Seite gehört die plugins-Sammlung von Internet Explorer zum document-Objekt und spiegelt im Wesentlichen die embeds-Sammlung wider: eine Sammlung aller embed-Elemente im Dokument. Ein embed-Element kann zwar ein Plug-In starten, dies ist jedoch nicht notwendigerweise der Fall. Darüber hinaus unterstützt Internet Explorer für Windows den JavaScript-Zugriff auf die installierten Plug-Ins in einer anderen Weise als Navigator (während IE für Macintosh ihn überhaupt nicht unterstützt).

Schreibgeschützt

Gibt die Anzahl von Elementen in der Sammlung zurück.
 
Beispiel
  
var IEhowMany = document.plugins.length;
var NNhowMany = navigator.plugins.length;
 
Wert

Eine ganze Zahl.

item(index[, subindex]) item(index)

Gibt ein einzelnes Objekt oder eine Objektsammlung zurück, das bzw. die dem mit dem Indexwert (oder optional mit dem Index- und dem Subindexwert) übereinstimmenden Element entspricht.

 
Parameter
 
  • Handelt es sich bei dem Parameter um eine auf Null basierende ganze Zahl, ist der zurückgegebene Wert ein einzelnes Element, das dem angegebenen Objekt in der Quellcodereihenfolge (verschachtelt im aktuellen Element) entspricht. Handelt es sich bei dem Parameter um eine Zeichenfolge (nur IE), ist der zurückgegebene Wert eine Sammlung der Elemente, deren id-Eigenschaft mit dieser Zeichenfolge übereinstimmt.
  • Wenn Sie eine Zeichenfolge als ersten Parameter angeben (nur IE), können Sie den zweiten Parameter verwenden, um einen auf Null basierenden Index anzugeben, der das angegebene Element aus einer Sammlung abruft, dessen name-Eigenschaft mit der Zeichenfolge des ersten Parameters übereinstimmt.
 
Zurückgegebener Wert

Ein Objekt oder eine Objektsammlung (Array). Falls keine Übereinstimmung mit den Parametern vorliegt, lautet der zurückgegebene Wert null.
