Das W3C-DOM-Objekt EventListener ist nichts weiter als eine Referenz auf eine Skriptfunktion, die ausgelöst durch ein Ereignis von einer Node aufgerufen wird. Seine Existenz im W3C-DOM bietet der Spezifikation eine bequeme Möglichkeit, den Datentyp des zweiten Parameters einer der weiter vorne in diesem Kapitel beschriebenen Methoden nodeObject.addEventListener( ) und nodeObject.removeEventListener( ) zu kennzeichnen.
