rowsNN 6 IE 4 DOM 1

Stellt eine Sammlung aller tr-Elementobjekte bereit, die in einem einzigen table-, tbody-, tfoot- oder thead-Elementobjekt enthalten sind. Die Objektsammlung rows eines table-Elements enthält alle Zeilen der Tabelle, ungeachtet der Zeilengruppen. Mitglieder dieser Sammlung werden in der Quellcodereihenfolge sortiert. Internet Explorer unterstützt die Array-Schreibweise und runde Klammern zum Zugriff auf ein einzelnes Element in der Sammlung (z. B. document.getElementByID("myTable).rows[0], document.all.myTable.rows (0)).

Gibt die Anzahl von Elementen in der Sammlung zurück.
 
Beispiel
  
var howMany = document.getElementById("myTable").rows.length;
 
Wert

Eine ganze Zahl.

item( )NN 6 IE 4 DOM 1

item(index[, subindex]) item(index)

Gibt ein einzelnes tr-Objekt oder eine tr-Sammlung zurück, das bzw. die dem mit dem Indexwert (oder bei IE optional mit dem index- und dem subindex-Wert) übereinstimmenden Element entspricht.

 
Parameter
 
  • Handelt es sich bei dem Parameter um eine auf Null basierende ganze Zahl, ist der zurückgegebene Wert ein einzelnes Element, das dem angegebenen Element in der Quellcodereihenfolge (verschachtelt im aktuellen Element) entspricht; handelt es sich bei dem Parameter eine Zeichenfolge, ist der zurückgegebene Wert eine Sammlung der Elemente, deren id-Eigenschaft mit der Zeichenfolge übereinstimmt.
  • Wenn Sie eine Zeichenfolge als ersten Parameter angeben (nur IE), können Sie den zweiten Parameter verwenden, um einen auf Null basierenden Index anzugeben, der das angegebene Element aus einer Sammlung abruft, deren id-Eigenschaft mit dem Zeichenfolgenwert des ersten Parameters übereinstimmt.
 
Zurückgegebener Wert

Ein tr-Objekt oder eine Sammlung (Array) mit tr-Objekten. Falls keine Übereinstimmung mit den Parametern vorliegt, lautet der zurückgegebene Wert null.

namedItem( )NN 6 IE 6 DOM 1

namedItem("ID")

Gibt ein einzelnes tr-Objekt oder eine tr-Sammlung zurück, das bzw. die dem mit dem Wert der Parameterzeichenfolge übereinstimmenden Element entspricht.

 
Parameter
 
  • Die Zeichenfolge, die denselben Wert wie das id-Attribut des gewünschten Elements enthält.
 
Zurückgegebener Wert

Ein tr-Objekt oder eine Objektsammlung (Array) mit tr-Objekten. Falls keine Übereinstimmung mit den Parametern vorliegt, lautet der zurückgegebene Wert null.

tags( )NN nicht zutreffend IE 4 DOM nicht zutreffend

tags(tagName)

Gibt eine Sammlung von Objekten zurück (aus allen in der aktuellen Sammlung enthaltenen Objekten), deren Tags mit dem tagName-Parameter übereinstimmen. In diesem Fall überflüssig, da alle Elemente dasselbe tr-Tag aufweisen.

 
Parameter
 
  • Eine Zeichenfolge mit der ausschließlich in Großbuchstaben geschriebenen Version des Element-Tags, z. B. document.getElementByID("myTable").rows.tags("tr").
 
Zurückgegebener Wert

Eine Objektsammlung (Array). Falls keine Übereinstimmung mit den Parametern vorliegt, ist der zurückgegebene Wert ein Array mit der Länge Null.
