Aufwärtsverbreitung: Nein (IE); Ja (NN 6); Abbruch möglich: Ja

Wird ausgelöst, wenn ein Benutzer eine Auswahl in ein Textfeld zieht (input- oder textarea-Elemente). Funktioniert in Navigator 4 für Windows nicht. In IE für Windows ab Version 4 gilt dieses Ereignis auch für ausgewählten Body-Text.

 
Typische Ziele

Text-input- und textarea-Elemente in allen Browsern, in IE für Windows ab Version 4 außerdem das body-Element.
