Das Entity-Objekt (einer der Nodetypen) ist im W3C-DOM eine abstrakte Darstellung eines Elements, das in einem XML-Dokument wie eine Speichereinheit behandelt wird. Einige Entitäten definieren einen Namen, der von anderen Elementen (einschließlich anderer Entitäten) als Kurzreferenz auf die in der Entität gespeicherten Informationen verwendet werden kann. Diese Referenz wird im W3C-DOM als EntityReference-Nodetyp dargestellt. In DTD-Dokumenten finden Sie zahlreiche Beispiele für solche Entity-Elemente. Weitere Informationen zur Anwendung von Entity-Elementen finden Sie in der XML-Spezifikation unter http://www.w3.org/TR/REC-xml.
