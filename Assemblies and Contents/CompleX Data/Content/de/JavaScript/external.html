externalNN nicht zutreffend IE 4(Win) DOM nicht zutreffend

Das external-Objekt wird im Wesentlichen von Entwicklern verwendet, die Internet Explorer als eine Komponente für ihre Anwendungen einsetzen und Zugriff auf angepasste Erweiterungen zum Dokumentobjektmodell benötigen. Sie können jedoch auch eine oder zwei Methoden dieses Objekts in Ihre Arbeiten für IE/Windows-Benutzer einbinden.

 

Der Code im folgenden Syntaxbeispiel fordert die Erlaubnis des Benutzers an, ein skriptgesteuertes Lesezeichen in die Favoritenliste des Browsers einzufügen: 

external.AddFavorite("URL", "Favorites List Label");
 

Der Code im folgenden Syntaxbeispiel lädt eine andere URL in ein Zielfenster oder einen Ziel-Frame (oder in das aktuelle Fenster, wenn im dritten Parameter eine leere Zeichenfolge übergeben wird) und führt anschließend eine Textsuche nach der im zweiten Parameter angegebenen Zeichenfolge durch: 

external.NavigateAndFind("URL", "searchString", "targetFrameName");
 

Weitere Informationen hierzu finden Sie unter http://msdn.microsoft.com/workshop/browser/overview/overview.asp.
