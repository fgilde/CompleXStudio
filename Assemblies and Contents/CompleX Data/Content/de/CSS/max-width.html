Vererbt: Nein

Mit diesen Attributen lässt sich für ein Element eine Mindest- bzw. Maximalbreite festlegen. Sie können die zulässige Breite eines Elements vorgeben, und zwar unabhängig davon, welche Breite sich aus dem normalen Inhaltsfluss innerhalb des übergeordneten Containers ergibt.

 
CSS-Syntax
  
max-width: length | percentage | none
min-width: length | percentage | none
 
Wert

Weitere Informationen hierzu finden Sie in der Diskussion über Längenwerte zu Beginn des Kapitels. Es kann auch ein Prozentwert relativ zum Container des Elements angegeben werden. Der Wert none bewirkt, dass alle Beschränkungen aufgehoben werden und ein natürlicher Inhaltsfluss möglich ist.

 
Anfangswert

none (max-width); none (min-width).

 
Gilt für

Alle Elemente.

 
Objektmodellreferenz
  
[window.]document.getElementById("elementID").style.minWidth
[window.]document.getElementById("elementID").style.maxWidth
