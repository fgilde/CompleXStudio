Diese Attribute steuern die Nummerierungsfolge eines für generierte Inhalte verwendeten CSS-Zählers (siehe Attribut content). Das Attribut counter-increment bestimmt, um welchen Wert (und in welche Richtung) sich der Zähler bei jedem Zugriff während der Darstellung ändert. Mit dem Attribut counter-reset können Sie den Zähler auf einen spezifischen Wert setzen (der Standardwert ist Null).

 
CSS-Syntax
  
counter-increment: counterID [ posOrNegInteger ] | none
counter-reset: counterID [ posOrNegInteger ] | none
 
Wert

counterID ist ein dem Attribut content:counter(counterID) zugewiesener Bezeichner. Der optionale ganzzahlige Wert folgt auf counterID und ist durch ein Leerzeichen getrennt. Sie können mehrere Zählerbezeichner in ein und demselben Stilattribut kombinieren, indem Sie durch Leerzeichen getrennte Bezeichner-/Ganzzahlenpaare aneinander reihen.

 
Anfangswert

none
 
Beispiel
  
h1 {counter-reset:subSection}
 
Gilt für

Alle Elemente.
