background-imageNN 4 IE 4 CSS 1

Vererbt: Nein

Legt das Hintergrundbild (falls zutreffend) für das Element fest. Wenn für das Element auch das Attribut background-color festgelegt wird, erscheint die Farbe, falls das Bild nicht geladen werden kann; andernfalls erscheint das Bild über der Farbe. Bei transparenten Pixeln des Bildes scheint die Hintergrundfarbe durch. Siehe auch das Attribut background-attachment.

 
CSS-Syntax
  
background-image: uri | none
 
Wert

Verwenden Sie zum Festlegen einer URL den Wrapper url( ) für den Attributwert. Sie können das Attribut weglassen oder none angeben, um das Laden eines Bildes in den Hintergrund des Elements zu verhindern.

 
Anfangswert

none
 
Beispiel
  
h1 {background-image: url(watermark.jpg)}
 
Gilt für

Alle Elemente.
 
Objektmodellreferenz
  
[window.]document.getElementById("elementID").style.backgroundImage
