border-bottom-color, border-left-color, border-right-color, border-top-colorNN 6 IE 4 CSS 2

Vererbt: Nein

Jedes der Attribute legt die Farbe einer einzelnen Rahmenseite eines Elements fest. Bei dieser Funktion ist darauf zu achten, dass nicht zusammengehörige Farben nicht vermischt werden. Informationen zum Festlegen der Farbe von mehreren Rahmenseiten in einer Anweisung finden Sie unter dem Attribut border-color.

 
CSS-Syntax
  
border-bottom-color: color
border-left-color: color
border-right-color: color
border-top-color: color
 
Wert

Weitere Informationen zum Wert color entnehmen Sie bitte dem Abschnitt über Farben am Anfang dieses Kapitels.

 
Anfangswert

Keiner.
 
Beispiel
  
p {border-bottom-color: gray}
div {border-left-color: #33c088}
p.special {border-right-color: rgb(150, 75, 0)}
h3 {border-top-color: rgb(100%, 50%, 21%)}
 
Gilt für

Alle Elemente (in IE 4 und 5 für Windows jedoch nur für Block- und ersetzte Elemente).

 
Objektmodellreferenz
  
[window.]document.getElementById("elementID").style.borderBottomColor
[window.]document.getElementById("elementID").style.borderLeftColor
[window.]document.getElementById("elementID").style.borderRightColor
[window.]document.getElementById("elementID").style.borderTopColor
