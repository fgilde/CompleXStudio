Kurzattribut, mit dem Sie in einer einzigen Anweisung ein oder mehr Layoutraster-Attribute (layoutGridChar, layoutGridLine, layoutGridMode und layoutGridType) festlegen können. Diese Attribute werden in erster Linie für asiatische Sprachen verwendet.

 
CSS-Syntax
  
layout-grid: layout-grid-mode | layout-grid-type | layout-grid-line |
layout-grid-char
 
Wert

Informationen zur Syntax und Beispiele für die Werte von Schrift- und Linienattributen finden Sie unter der jeweiligen Attributbeschreibung.

 
Anfangswert

both loose none none

 
Beispiel
  
body {layout-grid: both fixed 14px 14px}
 
Gilt für

Alle Elemente.
 
Objektmodellreferenz
  
[window.]document.getElementById("elementID").style.layoutGrid
