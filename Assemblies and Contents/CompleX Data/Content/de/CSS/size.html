Vererbt: --

Legt die Größe bzw. Orientierung einer Seitenbox fest. Diese Einstellungen dienen in erster Linie der Seitenformatierung für den Ausdruck und haben auf die Bildschirmanzeige möglicherweise keine Auswirkung. Dieses Attribut wird innerhalb einer @page-Deklaration gesetzt.

 
CSS-Syntax
  
size: [length {1,2}] auto | portrait | landscape
 
Wert

Wenn Sie ein oder zwei length-Werte angeben, wird die Größe der Seitenbox absolut und ohne Rücksicht auf die Papiergröße festgelegt; ohne spezifische length-Werte wird die Seitenbox dem ausgewählten Papierformat angepasst. Wenn Sie nur einen Längenwert angeben, wird dieser sowohl auf die Breite als auch auf die Höhe der Seitenbox angewendet; bei zwei Werten gilt der erste für die Breite, der zweite für die Höhe der Seitenbox. Beachten Sie, dass Drucker häufig einen Mindestrand um die dargestellte Seitenbox erzwingen. Auch wenn das Attribut size auf auto gesetzt ist, lässt sich um eine Seitenbox zusätzlicher Freiraum schaffen, indem das Attribut margin in die @page-Deklaration aufgenommen wird.

 
Anfangswert

auto
 
Beispiel
  
@page{size: landscape}
 
Gilt für

Seitenkontext.
