Vererbt: Nein

Legt fest, an welcher Seite der umgebenden Box das Element ausgerichtet wird bzw. wie andere Inhalte das Element umfließen. Bei der Einstellung none erscheint das Element in der Reihenfolge des Quellcodes, und höchstens eine Zeile des umgebenden Textes wird im gleichen horizontalen Band wie das Element angezeigt.

Da float früher als Schlüsselwort in JavaScript reserviert war, kann der Attributname in Objektmodellen, die JavaScript benutzen, nicht als Eigenschaft eines style-Objekts verwendet werden. Im Internet Explorer wird der Eigenschaftsname styleFloat verwendet, im W3C DOM und in Netscape 6 cssFloat.

 
CSS-Syntax
  
float: alignmentSide | none
 
Wert

Für alignmentSide besteht die Auswahl zwischen folgenden Konstanten: left | right.

 
Anfangswert

none
 
Beispiel
  
img.navButton {float: right}
 
Gilt für

Alle Elemente mit Ausnahme von positionierten Elementen (oder generierten Inhalten).
 
Objektmodellreferenz
  
[window.]document.getElementById("elementID").style.styleFloat
[window.]document.getElementById("elementID").style.cssFloat
