Definiert bei positionierbaren Elementen die Position der linken Kante einer Elementbox (Inhalt plus linke Auffüllung, Rahmen und/oder Ränder) relativ zur linken Kante des nächst äußersten Blockinhaltscontainers. Wenn das Element relativ positioniert ist, bezieht sich der Offset auf die linke Kante der Inline-Position, an der das Element normalerweise im Inhalt erscheinen würde.

 
CSS-Syntax
  
left: length | percentage | auto
 
Wert

Weitere Informationen hierzu finden Sie in der Diskussion über Längenwerte zu Beginn dieses Kapitels. Negative Längen können in bestimmten Fällen erlaubt sein, doch sollten Sie dies stets in allen Browsern testen. Auch die Eingabe eines Prozentwerts ist möglich; dieser wird ausgehend von der Breite des nächstäußeren Containers berechnet. Bei der Einstellung auto kann der Browser den linken Offset der Elementbox innerhalb des umgebenden Rechtsecks anhand des normalen Elementflusses bestimmen. Beim Navigator wird das Element sehr nahe an der linken Kante der umgebenden Box dargestellt, wohingegen beim Internet Explorer ein kleiner Rand bleibt.

 
Anfangswert

auto
 
Beispiel
  
h1 {position: relative; left: 2em}
#logo {position: absolute; left: 80px; top: 30px}
 
Gilt für

Positionierte Elemente.
 
Objektmodellreferenz
  
[window.]document.getElementById("elementID").style.left
