ruby-overhangNN nicht zutreffend IE 5 CSS 3

Vererbt: Ja

Steuert den Textüberhang von Inhalt in einem ruby-Element.

 
CSS-Syntax
  
ruby-overhang: alignType | auto
 
Wert

Eine der folgenden Konstanten: auto | none | whitespace. Weitere Einzelheiten zu Ruby-spezifischen Stilen finden Sie unter http://www.w3.org/TR/css3-ruby.

 
Anfangswert

auto
 
Gilt für

ruby-Elemente (und andere Elemente, bei denen das Attribut display auf ruby-text gesetzt ist).

 
Objektmodellreferenz
  
[window.]document.getElementById("elementID").style.rubyOverhang
