Nur für aurale Stylesheets; ein Cue ist ein Hilfston (auch als Hörsymbol bezeichnet), mit dem beim Vorlesen Dokumentinhalte hörbar abgegrenzt werden können. Die Attribute cue-before und cue-after sind URIs von Audiodateien, die vor bzw. nach der Wiedergabe von Inhalten über ein entsprechendes Medium abgespielt werden.

 
CSS-Syntax
  
cue-after: uri | none
cue-before: uri | none
 
Wert

Jede gültige komplette oder relative URL (im CSS-Format) einer Audiodatei mit einem vom Browser unterstützten MIME-Typ. Sie können beiden Attributen den gleichen Wert zuweisen, falls dies sinnvoll ist.

 
Anfangswert

none
 
Beispiel
  
li {cue-before: url(ding.wav); cue-after: url(dong.wav)}
 
Gilt für

Alle Elemente.
