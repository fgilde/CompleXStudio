Steuert die Opazität des Elements. Je niedriger der Wert, desto durchsichtiger das Element. Dies ist die Mozilla-spezifische Version des Microsoft-spezifischen Opazitätsfilters.

 
CSS-Syntax
  
-moz-opacity: alphaValue
 
Wert

Die Opazität wird durch eine Fließkommazahl zwischen 0.0 und 1.0 bestimmt. Beim Wert 1.0 wird das Element völlig undurchsichtig dargestellt. Sie können auch einen Prozentwert zwischen 0% und 100% angeben, obwohl die vorgeschlagene CSS3-Empfehlung für das Attribut opacity nur Zahlenwerte vorsieht.

 
Anfangswert

1
 
Beispiel
  
div#watermark {-moz-opacity:0.4}
 
Gilt für

Alle Elemente.
 
Objektmodellreferenz
  
[window.]document.getElementById("elementID").style.MozOpacity
