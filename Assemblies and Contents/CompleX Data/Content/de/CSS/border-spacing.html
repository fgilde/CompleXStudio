Vererbt: Ja

Legt die Größe des Abstands (falls zutreffend) zwischen allen Zellenrahmen in einer Tabelle fest. Für dieses Attribut muss das Attribut border-collapse auf separate gesetzt sein (normalerweise der Standardwert). Falls Sie nur einen Längenwert übergeben, wird dieser sowohl für den horizontalen als auch für den vertikalen Abstand verwendet; bei zwei Werten gilt der erste für die horizontale und der zweite für die vertikale Richtung. In Abbildung 11-2 finden Sie eine Übersicht über die verschiedenen Maßdefinitionen für Tabellen.

 
CSS-Syntax
  
border-spacing: length[length]
 
Wert

Weitere Informationen hierzu finden Sie in der Diskussion über Längenwerte zu Beginn dieses Kapitels. Falls Sie entlang einer Achse keinen Abstand wünschen, setzen Sie den entsprechenden Wert auf Null.

 
Anfangswert

0

table-Element.
