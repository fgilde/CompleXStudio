Ermöglicht einem Element, die x-Höhe (in "ex" gemessen) einer bestimmten Schrift beim Ersetzen von Schriften beizubehalten. Der z-Faktor ist das Verhältnis der em- zur x-Höhe einer Schrift. Da verschiedene Schriften der gleichen Größe auf einer Seite unterschiedlich groß aussehen können, kann mit dem z-Faktor das Verhältnis berechnet und auf andere Schriften angewendet werden. Auch wenn die resultierende Schriftgröße auf diese Weise größer oder kleiner als bei der Referenzschrift sein kann, stimmen die Größen subjektiv besser überein. Dies hat normalerweise auch zur Folge, dass die Wortlängen bei verschiedenen Schriftfamilien sich angleichen und Zeilenumbrüche identisch vorgenommen werden.

 
CSS-Syntax
  
font-size-adjust: 0.47
 
Wert

Eine Zahl für das Seitenverhältnis der bevorzugten Schrift (u. U. beim Schriftersteller erfragbar) oder none.

 
Anfangswert

none
 
Gilt für

Alle Elemente.

 
Objektmodellreferenz
  
[window.]document.getElementById("elementID").style.fontSizeAdjust
