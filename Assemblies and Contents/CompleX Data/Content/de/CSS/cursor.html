Legt die Form des Cursors fest, wenn sich dieser über dem Element befindet. Das genaue Erscheinungsbild des Cursors hängt vom Betriebssystem ab. Bevor Sie einen modifizierten Cursor verwenden, sollten Sie die Funktionen der verschiedenen Cursortypen im Browser und im Betriebssystem genau kennen. Benutzer erwarten, dass ein bestimmtes Cursordesign in verschiedenen Anwendungen die gleiche Bedeutung hat. In Abbildung 11-3 sind die Cursor für die einzelnen Cursor-Konstanteneinstellungen in Internet Explorer für Windows dargestellt.

 
CSS-Syntax
  
cursor: cursorType || uri
 
Wert

Ein Cursortyp ist einer der implementierten Cursornamen. Aus der folgenden Tabelle geht hervor, welche Cursortypen von den verschiedenen Browsern und dem CSS-Standard unterstützt werden.

In IE 6 für Windows sind herunterladbare Cursor implementiert. Die IE 6-Einstellung für eine externe URL erfordert die Adresse einer Cursordatei mit der Erweiterung .cur oder .ani (die Sie mit einem Grafikprogramm erstellen, das Windows-Cursor unterstützt).
