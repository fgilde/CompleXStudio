z-indexNN 4 IE 4 CSS 2

Vererbt: Nein

Legt für ein positioniertes Element die Stapelreihenfolge relativ zu anderen Elementen im selben übergeordneten Container fest. Einzelheiten zu den Beziehungen zwischen Elementebenen in mehreren Containern finden Sie in Kapitel 4.

 
CSS-Syntax
  
z-index: integer | auto
 
Wert

Beliebiger Integerwert. Die Einstellung auto entspricht dem Wert Null. Wenn für alle Elemente in einem übergeordneten Container der gleiche z-index-Wert definiert ist, wird die Reihenfolge entsprechend der Abfolge im Quellcode bestimmt.

 
Anfangswert

auto
 
Beispiel
  
div#instrux {position: absolute; left: 50; top: 70; z-index: 2}
 
Gilt für

Positionierte Elemente.
 
Objektmodellreferenz
  
[window.]document.getElementById("elementID").style.zIndex
