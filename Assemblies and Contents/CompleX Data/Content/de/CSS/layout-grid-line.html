layout-grid-lineNN nicht zutreffend IE 5(Win) CSS nicht zutreffend

Vererbt: Ja

Steuert bei asiatischen Sprachen die Zeilenhöhe des Zeichenrasters für Blockebenen-Elemente.

 
CSS-Syntax
  
layout-grid-line: length | auto | none
 
Wert

Längenwert als absoluter oder Prozentwert bzw. eine der folgenden Konstanten: auto | none.

 
Anfangswert

none

 
Beispiel
  
body {layout-grid-mode:both; layout-grid-line: 14px}
 
Gilt für

Alle Elemente.
 
Objektmodellreferenz
  
[window.]document.getElementById("elementID").style.layoutGridLine
