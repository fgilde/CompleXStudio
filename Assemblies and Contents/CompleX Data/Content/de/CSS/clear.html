Definiert, ob ein Blockebenen-Element im selben horizontalen Band wie ein nahe gelegenes schwebendes Element, etwa ein Bild, dargestellt werden kann. In der Regel ist für ein anderes Element in der Nähe das float-Stilattribut auf left oder right gesetzt. Um zu verhindern, dass das aktuelle Element im gleichen Band wie der unverankerte Block erscheint, setzen Sie das clear-Attribut auf die gleiche Seite (left oder right). Wenn Sie nicht sicher sind, wo eine potenzielle Überlappung auftreten könnte, setzen Sie das clear-Attribut auf both. Ein Element, dessen clear-Attribut auf einen anderen Wert als none gesetzt wurde, wird am Anfang der nächsten verfügbaren Zeile unter dem unverankerten Element dargestellt.

 
CSS-Syntax
  
clear: both | left | none | right
 
Wert

Eine der folgenden Konstanten: both | left | none | right.

 
Anfangswert

none
 
Beispiel
  
<img src="logo.gif" height="40" width="60" style="float: right">
<h1 style="clear: right">Giantco Corporation</h1>
 
Gilt für

Blockebenen-Elemente.
