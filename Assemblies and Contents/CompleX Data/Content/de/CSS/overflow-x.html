overflow-x, overflow-yNN nicht zutreffend IE 5(Win) CSS nicht zutreffend

Vererbt: Nein

Bestimmt, wie das Element Inhalt behandelt, dessen Abmessungen bei der Darstellung die angegebene Breite (x) oder Höhe (y) des Containers überschreiten. Dieses IE/Windows-Attribut funktioniert im Prinzip wie das normale overflow-Attribut, wirkt sich aber nur auf eine Achse aus. Es ist besonders nützlich, wenn nur eine vertikale oder nur eine horizontale Bildlaufleiste zusammen mit einem Element erscheinen soll. Weitere Informationen finden Sie in der Beschreibung des Attributs overflow.

 
CSS-Syntax
  
overflow-x: overFlowType
overflow-y: overFlowType
 
Wert

Eine der folgenden Konstanten: auto | hidden | scroll | visible.

 
Anfangswert

visible
 
Beispiel
  
body {overflow-x:hidden; overflow-y: scroll}
 
Gilt für

Blockebenen-, ersetzte und positionierte Elemente.
 
Objektmodellreferenz
  
[window.]document.getElementById("elementID").style.overflowX
[window.]document.getElementById("elementID").style.overflowY
