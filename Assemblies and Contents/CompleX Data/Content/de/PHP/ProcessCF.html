pcntl_exec PHP 4.1.0

bool pcntl_exec(string path[, array args[, array envs]])

  

Führt das angegebene Programm im aktuellen Prozessbereich aus, wie durch exec() definiert.

pcntl_fork PHP 4.1.0

int pcntl_fork(void)

 

Führt für den aktuellen Prozess wie beim Unix-Systemaufruf fork( ) ein Forking durch.

pcntl_signal PHP 4.1.0

bool pcntl_signal(long signo, mixed handle)

 

Weist einer PHP-Funktion einen Systemsignalhandler zu.
pcntl_waitpid PHP 4.1.0

int pcntl_waitpid(long pid, long status, long options)

 

Wartet auf den Status eines geforkten Childs bzw. gibt ihn zurück, wie durch den Systemaufruf waitpid( ) definiert.

pcntl_wexitstatus PHP 4.1.0

int pcntl_wexitstatus(long status)

 

Gibt den Statuscode zurück, der bei Beendigung eines Childs generiert wurde.
pcntl_wifexited PHP 4.1.0

bool pcntl_wifexited(long status)

  

Gibt true zurück, wenn der Statuscode für das Child angibt, dass dieses erfolgreich beendet wurde.
pcntl_wifsignaled PHP 4.1.0

bool pcntl_wifsignaled(long status)

  

Gibt true zurück, wenn der Statuscode für das Child einen Prozess angibt, der aufgrund eines Signals beendet wurde.

pcntl_wifstopped PHP 4.1.0

bool pcntl_wifstopped(long status)

 

Gibt true zurück, wenn der Statuscode für das Child einen angehaltenen Prozess angibt (waitpid() muss mit WUNTRACED aufgerufen worden sein).

pcntl_wstopsig PHP 4.1.0

int pcntl_wstopsig(long status)

  

Gibt die Nummer des Signals an, durch das der angegebene Prozess angehalten wurde.

pcntl_wtermsig PHP 4.1.0

int pcntl_wtermsig(long status)

  

Gibt die Nummer des Signals an, durch das der angegebene Prozess beendet wurde.
