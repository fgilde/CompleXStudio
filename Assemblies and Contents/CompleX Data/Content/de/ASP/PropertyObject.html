Property-Objekt (Versionen 2.0, 2.1, 2.5 und 2.6)
Dim property As ADODB.Property

Das Objekt Property enthält Informationen zu dynamischen Eigenschaften, die vom zugeordneten Datenprovider implementiert werden.

Property.Attributes-Eigenschaft (Versionen 2.0, 2.1, 2.5 und 2.6)

property.Attributes = PropertyArgumentsEnum
 

Die Eigenschaft Attributes legt bestimmte Merkmale eines Property-Objekts fest bzw. gibt sie zurück.
Datentyp

PropertyArgumentsEnum (Long)
 
Beschreibung

Die Eigenschaft Attributes ist schreibgeschützt. Der Wert der Eigenschaft Arguments kann auf eine beliebige Summe der in der Enumeration PropertyArgumentsEnum in Anhang E aufgeführten Werte gesetzt werden.

PropertyAttributesEnum-Enumeration
 
Property.Name-Eigenschaft (Versionen 2.0, 2.1, 2.5 und 2.6)

property.Name = name
 

Die Eigenschaft Name gibt den Namen des aktuellen Property-Objekts an.
Datentyp

String
 
Beschreibung

Die Eigenschaft Name ist für das Property-Objekt schreibgeschützt. Diese Eigenschaft ruft ein Property-Objekt aus der Kollektion Properties ab. Innerhalb der Kollektionsklasse Properties müssen Namen nicht eindeutig sein.
 
Property.Type-Eigenschaft (Versionen 2.0, 2.1, 2.5 und 2.6)

type = property.Type
 

Die Eigenschaft Type gibt den Datentyp des Werts des Property-Objekts an.
Datentyp

DataTypeEnum
 
Beschreibung

Die Eigenschaft Type ist für Property-Objekte schreibgeschützt.

Property.Value-Eigenschaft (Versionen 2.0, 2.1, 2.5 und 2.6)

value = property.Value
 

Die Eigenschaft Value gibt den Wert des aktuellen Property-Objekts an.
Datentyp

Variante
 
Beschreibung

Legt den Wert des aktuellen Property-Objekts fest bzw. gibt ihn zurück.
