<jsp:getProperty>  

Die Aktion <jsp:getProperty> fügt den Wert einer Bean-Eigenschaft, die in einen String konvertiert wurde, der Antwort hinzu, die von der Seite erzeugt wurde.

Beispiel:

<jsp:getProperty name="clock" property="hours" />
name String
Wert zur Anforderungszeit akzeptiert: nein

Obligatorisch. Der Name, der einem Bean in einer JSP-Ebene zugeordnet ist.

property String
Wert zur Anforderungszeit akzeptiert: nein

Obligatorisch. Der Name der Bean-Eigenschaft, die in die Seite eingeschlossen werden soll.
