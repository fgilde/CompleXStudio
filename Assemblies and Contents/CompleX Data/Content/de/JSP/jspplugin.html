<jsp:plugin>  

Die Aktion <jsp:plugin> generiert HTML-Elemente <embed> oder <object> (je nach Browsertyp). Diese HTML-Elemente bewirken den Download der Java Plug-In-Software (falls erforderlich) und die darauf folgende Ausführung des angegebenen Java-Applets oder der angegebenen JavaBeans-Komponente. Der Body der Aktion kann ein Element <jsp:params> enthalten, das Applet-Parameter angibt, sowie ein Element <jsp:fallback>, das den Text angibt, der in Browsern angezeigt wird, die die HTML-Elemente <embed> oder <object> nicht unterstützen. Weitere Informationen zum Java Plug-In finden Sie unter http://java.sun.com/products/plugin/.

Beispiel:

<jsp:plugin type="applet" code="Clock2.class" 
  codebase="applet" 
  jreversion="1.2" width="160" height="150" >
  <jsp:params>
    <jsp:param name="bgcolor" value="ccddff" />
  </jsp:params>
  <jsp:fallback>
    Plug-in tag OBJECT or EMBED not supported by 
    browser.
  </jsp:fallback>
</jsp:plugin>
align String
Wert zur Anforderungszeit akzeptiert: nein

Optional. Die Ausrichtung des Applet-Bereichs, entweder bottom, middle oder top.

archive String
Wert zur Anforderungszeit akzeptiert: nein

Optional. Eine kommagetrennte Liste von URIs für Archive mit Klassen und anderen Ressourcen, die im Voraus geladen werden. Zum Laden dieser Klassen wird eine Instanz von AppletClassLoader mit der angegebenen codebase verwendet. Relative URIs für Archive werden in Bezug auf die codebase des Applets interpretiert.

code String
Wert zur Anforderungszeit akzeptiert: nein

Obligatorisch. Der vollständige Klassenname des Objekts.

codebase String
Wert zur Anforderungszeit akzeptiert: nein

Obligatorisch. Die relative URL für das Verzeichnis, das die Klassendatei enthält. Gemäß der HTML 4.0-Spezifikation muss es sich beim Verzeichnis um ein Unterverzeichnis des Verzeichnisses handeln, das die Seite enthält.

height String
Wert zur Anforderungszeit akzeptiert: nein

Optional. Die Höhe des Applet-Bereichs in Pixel oder als Prozentwert.

hspace String
Wert zur Anforderungszeit akzeptiert: nein

Optional. Die Größe des Leerraums, der links und rechts vom Applet-Bereich eingefügt werden soll, und zwar in Pixel.

iepluginurl String
Wert zur Anforderungszeit akzeptiert: nein

Optional. Die URL des Java Plug-Ins für Internet Explorer. Der Standardwert richtet sich nach der Implementierung.

jreversion String
Wert zur Anforderungszeit akzeptiert: nein

Optional. Die Versionsnummer von JRE, die die Komponente zum Betrieb benötigt. Der Standardwert ist 1.1.

name String
Wert zur Anforderungszeit akzeptiert: nein

Optional. Der Applet-Name, der von anderen Applets auf derselben Seite zur Kommunikation mit dem Applet verwendet wird.

nspluginurl String
Wert zur Anforderungszeit akzeptiert: nein

Optional. Die URL des Java Plug-Ins für Netscape. Der Standardwert richtet sich nach der Implementierung.

title String
Wert zur Anforderungszeit akzeptiert: nein

Optional. Der Text, den der Browser für das Applet anzeigen soll (wie "Quickinfo").

type String
Wert zur Anforderungszeit akzeptiert: nein

Obligatorisch. Der Typ des einzubettenden Objekts, entweder applet oder bean.

vspace String
Wert zur Anforderungszeit akzeptiert: nein

Optional. Die Größe des Leerraums, der oberhalb und unterhalb vom Applet-Bereich eingefügt werden soll, und zwar in Pixel.

width String
Wert zur Anforderungszeit akzeptiert: nein

Optional. Die Breite des Applet-Bereichs in Pixel oder als Prozentwert.
