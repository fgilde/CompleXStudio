Beschreibung

Die Variable exception wird der Unterklasse von Throwable zugewiesen, die das Aufrufen der Fehlerseite verursacht hat. Die Klasse Throwable ist die übergeordnete Klasse aller Fehler und Ausnahmen in der Java-Programmiersprache. Nur Objekte, die Instanzen dieser Klasse (oder einer ihrer Unterklassen) sind, werden von Java Virtual Machine ausgelöst oder können von der Java-Anweisung throw ausgelöst werden. In der Java-Dokumentation unter http://java.sun.com/docs/index.html finden Sie eine Beschreibung der Klasse Throwable.
