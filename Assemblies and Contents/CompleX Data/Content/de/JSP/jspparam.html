<jsp:param>  

Sie können die Aktion <jsp:param> im Body einer Aktion <jsp:forward> oder <jsp:include> verwenden, um zusätzliche Anforderungsparameter für die Zielressource festzulegen. Sie kann auch im Body einer Aktion <jsp:params> verwendet werden, um Applet-Parameter festzulegen.

Beispiel:

<jsp:include page="navigation.jsp">
  <jsp:param name="bgColor" value="<%= currentBGColor %>"
    />
</jsp:include>
name String
Wert zur Anforderungszeit akzeptiert: nein

Obligatorisch. Der Parametername.

value String
Wert zur Anforderungszeit akzeptiert: ja

Obligatorisch. Der Parameterwert.
